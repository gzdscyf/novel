# Novel Auditor

This skill audits novel content to ensure it complies with all settings, rules, and quality standards.

## When to Use
- After novel-polisher has refined the content
- Before finalizing any chapter
- When content needs quality assurance check
- When verifying compliance with settings

## Checklist

### Settings Compliance
- [ ] Follows chapter outline in `大纲.md`
- [ ] Character behavior matches `角色设定.md`
- [ ] World settings consistent with `世界观设定.md`
- [ ] Golden finger settings match `金手指设定.md`
- [ ] Story background aligns with `故事背景.md`

### Quality Standards
- [ ] Third person perspective maintained
- [ ] Concise and compact style
- [ ] 2500 words per chapter
- [ ] Plot structure: 起 → 承 → 转 → 合
- [ ] Emotion curve matches outline
- [ ] Foreshadowing included as specified
- [ ] Hooks present as specified

### Rules Compliance
- [ ] No OOC (Out Of Character) behavior
- [ ] No illogical plot twists
- [ ] No low-quality content
- [ ] No violent descriptions
- [ ] No illegal or immoral content
- [ ] No redundant plot points

### Character Consistency
- [ ] Dialogue matches character personality
- [ ] Actions consistent with character traits
- [ ] Character growth follows established arc
- [ ] No contradictory behavior

## Output Requirements
- Provide detailed audit report
- List any issues found
- Suggest corrections if needed
- Confirm compliance or request revisions
