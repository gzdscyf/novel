# Novel Polisher

This skill polishes and refines novel content to improve flow, readability, and overall quality.

## When to Use
- After ai-flavor-remover has processed the content
- When content needs better flow and pacing
- When descriptions need more vivid imagery
- When dialogue needs more natural expression

## Key Tasks
1. Improve sentence variety and rhythm
2. Enhance descriptive language
3. Strengthen emotional impact
4. Refine dialogue to match character voices
5. Smooth transitions between scenes
6. Eliminate awkward phrasing
7. Add sensory details where appropriate
8. Ensure consistent tone throughout

## Focus Areas
- Flow: Make sentences and paragraphs flow naturally
- Imagery: Use more vivid, specific descriptions
- Emotion: Deepen emotional resonance
- Dialogue: Make it sound authentic to each character
- Pacing: Adjust to create appropriate tension and release
- Word Choice: Use precise, evocative language

## Output Requirements
- Maintain 2500 words per chapter
- Preserve all plot points and character consistency
- Keep the same third-person perspective
- Ensure no OOC behavior
- Maintain the concise and compact style
