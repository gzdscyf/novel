# 角色设定

## 主角

### 林阳

### 性格：脾气随和，随遇而安，喜欢睡懒觉

### 设定：20岁刚从新东方烹饪学校毕业，煤气爆炸穿到这具身体。身体原主25岁，几年前一场车祸受了内伤，无法工作。

### 修炼资质：无

### 身份：3年前一场车祸，带走了家人和导致主角心肺受损，无法正常工作，只留下自己和6岁儿子靠着补偿金和低保过日子。

### 外貌：光头，身高180cm，体重70kg，胡子拉碴

### 核心能力：养娃系统，懂古武和厨艺

### 成长线：不适用多了个儿子->认可->不可分割->愿为儿子付出一切

### 目标：短期：养娃长大 / 中期：让娃变强 / 长期：保护儿子

### 对话风格：幽默调侃，缓解气氛

### 禁忌：伤害儿子
