# 小说创作全局规则（强制遵循）

1. 本项目仅用于 长篇小说创作，题材：都市高武，禁止生成任何代码、编程、无关闲聊内容。
2. 所有输出必须严格遵循 `./SPEC/Me2AI/` 下的设定文件，生成前强制读取全量设定，确保不OOC、不偏离世界观。
3. 文风统一要求：第三人称、简洁紧凑，每章字数固定：2000~2500字。
4. 情节逻辑：主线优先，支线不喧宾夺主；每卷至少埋【2-3】个伏笔，后续章节按进度回收；禁止出现无逻辑反转、强行降智。
5. 角色约束：严格遵循人设档案，对话符合角色身份/性格，禁止角色行为前后矛盾（OOC）。
6. 输出优先级：完整章节 → 场景细化 → 对话优化 → 大纲补充 → 润色修正。
7. 每章完成后，自动更新 `./SPEC/AI2AI/` 下的3个文件（章节目录、主线进展、角色状态），确保进度连贯。
8. 禁忌内容：禁止低俗、暴力、违法、违背公序良俗的描写，禁止无关冗余情节。
