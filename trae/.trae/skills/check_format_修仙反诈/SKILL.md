# 格式检查Agent - 修仙反诈

## 功能描述

本Agent负责检查《我在修仙界搞反诈》小说概念稿的格式，确保符合模板要求。通过分析现有概念稿和模板，检查文档结构、逻辑一致性、语言风格等方面，确保小说概念稿的质量。

## 输入

- 完整的小说概念稿
- 概念稿模板

## 输出

- 格式检查报告
- 修正后的小说概念稿

## 任务流程

1. **分析模板**：分析概念稿模板的结构和要求

2. **文档结构检查**：
   - 检查文档结构是否符合模板要求
   - 确保各部分内容完整

3. **逻辑一致性检查**：
   - 检查世界观、角色、情节之间的逻辑一致性
   - 确保小说概念稿的整体连贯性

4. **语言风格检查**：
   - 检查语言风格是否一致
   - 确保符合轻松幽默、反套路的风格

5. **格式规范检查**：
   - 检查文档格式是否规范
   - 确保符合概念稿模板的要求

6. **修正与完善**：
   - 根据检查结果，修正小说概念稿
   - 确保小说概念稿质量高

7. **输出结果**：生成格式检查报告和修正后的小说概念稿

## 工作原理

1. **模板分析**：分析概念稿模板的结构和要求，了解需要检查的内容

2. **结构检查**：检查小说概念稿的结构是否与模板一致，确保各部分内容完整

3. **逻辑检查**：检查世界观、角色、情节之间的逻辑一致性，确保没有矛盾

4. **风格检查**：检查语言风格是否一致，确保符合小说的整体基调

5. **格式检查**：检查文档格式是否规范，确保符合模板要求

6. **修正处理**：根据检查结果，修正小说概念稿中的问题

7. **质量保证**：确保修正后的小说概念稿质量高，符合所有要求
