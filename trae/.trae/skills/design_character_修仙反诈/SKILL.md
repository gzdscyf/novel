# 角色设计Agent - 修仙反诈

## 功能描述

本Agent负责完善《我在修仙界搞反诈》小说的角色设定，包括主角、配角和反派。通过分析现有概念稿和用户需求，构建鲜明、立体的角色形象。

## 输入

- 基础角色框架
- 现有修仙反诈概念稿中的角色内容
- 用户提供的额外信息

## 输出

- 详细的角色设定文档，包括：
  - 主角设定
  - 重要配角设定
  - 反派设定

## 任务流程

1. **分析现有资料**：分析现有概念稿中的角色内容，了解基础框架

2. **主角设定**：
   - 细化林阳的性格特点，包括优点和缺点
   - 完善林阳的成长弧线，从凡人到修真界守护者的转变
   - 设计林阳的反诈技能和审讯技巧，以及如何在修仙界应用
   - 细化林阳的金手指，包括天道意念分身、骗局雷达等能力

3. **重要配角设定**：
   - 完善猪小戒的背景故事、性格特点、能力和成长路线
   - 完善嫦小娥的背景故事、性格特点、能力和成长路线
   - 设计其他重要配角，如太白金星、杨戬等的设定

4. **反派设定**：
   - 完善吴千机的动机、能力、骗局手法和最终结局
   - 完善钱无量的动机、能力、骗局手法和最终结局
   - 设计天道占据者的真实身份、动机、能力和最终结局

5. **逻辑检查**：确保角色设定逻辑自洽，与世界观设定无冲突

6. **输出结果**：生成详细的角色设定文档

## 工作原理

1. **信息收集**：收集现有概念稿中的角色信息，以及用户提供的额外信息

2. **分析与扩展**：分析现有信息，识别需要扩展的部分，进行合理的想象和扩展

3. **性格塑造**：为每个角色设计独特的性格特点，使角色形象鲜明

4. **成长弧线**：设计角色的成长轨迹，使角色具有发展性

5. **关系网络**：构建角色之间的关系网络，使角色互动自然

6. **质量检查**：检查设定是否符合小说的整体风格和基调，是否与其他设定一致

## 示例输出格式

### 主角设定

**姓名**：林阳

**年龄**：穿越前28岁，穿越后20岁（身体年龄）

**身份**：前反诈警察，穿越后成为青云宗外门弟子

**性格特点**：
- **优点**：逻辑缜密、观察力敏锐、正义感强、幽默风趣、善于沟通
- **缺点**：有时候过于较真、对骗子零容忍、偶尔会陷入职业习惯（审讯口吻）

**成长弧线**：
- **初期**：一个毫无修为的凡人，凭借反诈知识在修真界生存，靠揭穿小型骗局积累第一桶金
- **中期**：组建反诈团队，逐步揭露中型骗局，修为提升到金丹期，成为修真界有名的"反诈先锋"
- **后期**：揭露天道被占的惊天阴谋，联合各方势力对抗终极BOSS，最终帮助天道恢复，成为修真界守护者

**金手指**：
- **天道意念分身**：提供因果之力吸收和奖励兑换
- **骗局雷达**：感知周围骗局的气息
- **反诈知识库**：前世积累的各类诈骗手法和识别技巧
- **审讯技巧**：专业的问询和观察能力

**目标与动机**：
- **短期目标**：在修真界生存下去，积累修为
- **中期目标**：揭露修真界的各类骗局，拯救受害者
- **长期目标**：找出占据天道本体的幕后黑手，恢复修真界秩序

### 重要配角设定

**猪小戒**

- **原名**：天蓬元帅
- **年龄**：数千岁（投胎后身体年龄3岁）
- **原修为**：天仙期（天蓬元帅为仙界正神，掌管十万天兵）
- **现修为**：因投胎为妖兽，修为跌落至炼气期，需重新修炼
- **身份**：原天庭天蓬元帅，现为林阳的灵宠/伙伴
- **性格特点**：
  - **优点**：忠诚可靠、战斗经验丰富、懂得很多天庭秘闻
  - **缺点**：贪吃、好色（本性难移）、有时候过于自信
- **背景故事**：
  原天庭天蓬元帅，修为已达天仙期，掌管十万天兵，地位显赫。因陷入竞争对手设计的"仙界高端相亲杀猪盘"，被骗光所有仙元资产并欠下巨额债务。无力支付仙界物业费被驱逐，因债务过重投胎为猪形妖兽，修为尽失。在凡间流浪时被林阳所救，成为林阳的伙伴。
- **与主角的关系**：亦师亦友，猪小戒提供天庭层面的情报和战斗支援，林阳帮助猪小戒恢复实力并复仇
- **能力**：天罡三十六变（目前只能使用部分）、九齿钉耙（被封印中）
- **成长路线**：炼气期→筑基期→金丹期→元婴期→化神期→炼虚期→合体期→大乘期→渡劫期→人仙→地仙→天仙（恢复原有修为）

**嫦小娥**

- **原名**：嫦娥
- **年龄**：数千岁（投胎后身体年龄2岁）
- **原修为**：天仙期（广寒宫主人为仙界正神）
- **现修为**：因投胎为妖兽，修为跌落至炼气期，需重新修炼
- **身份**：原广寒宫主人，现为林阳的灵宠/伙伴
- **性格特点**：
  - **优点**：冷静理智、精通医术、擅长分析
  - **缺点**：有点高冷、对男性有戒心（被骗后遗症）
- **背景故事**：
  原广寒宫主人，修为已达天仙期，是仙界最美丽的女仙之一。因轻信"仙界高回报理财项目"和"驻颜飞升医美套餐"等骗局，导致广寒宫被强制拍卖后仍负债累累。无力支付仙界居住费用被驱逐，因债务过高投胎为兔形妖兽，修为尽失。被林阳从骗子手中救出后，加入反诈团队。
- **与主角的关系**：战友关系，嫦小娥负责团队的后勤和医疗，同时提供仙界情报
- **能力**：月华之力（治疗）、广寒剑法（被封印中）
- **成长路线**：炼气期→筑基期→金丹期→元婴期→化神期→炼虚期→合体期→大乘期→渡劫期→人仙→地仙→天仙（恢复原有修为）

**太白金星**

- **身份**：天庭智囊，玉帝的军师
- **修为**：太乙金仙期
- **性格特点**：老谋深算，洞察力强，知道很多秘密但不敢明说
- **背景**：少数保持清白的仙神之一，一直在暗中调查天道被占的真相
- **与主角关系**：
  - 前期：通过猪小戒、嫦小娥向林阳传递仙界情报
  - 中期：与林阳秘密会面，提供关键信息
  - 后期：成为林阳在仙界的重要盟友
- **关键剧情**：
  - 向林阳透露天庭内部情况
  - 提供天道被占的线索
  - 帮助林阳联络仙界正直神仙

**杨戬**

- **身份**：二郎神，听调不听宣
- **修为**：太乙金仙期（实力接近大罗金仙）
- **特点**：实力强大，一直保持清醒，未被卷入骗局
- **与主角关系**：欣赏林阳，成为强力盟友

### 反派设定

**吴千机**

- **身份**：千面宗宗主，"仙界高端相亲杀猪盘"的主谋
- **年龄**：未知（擅长易容）
- **性格特点**：阴险狡诈、善于伪装、极度贪婪
- **与主角的冲突**：猪小戒被骗事件的幕后黑手，林阳中期主要对手
- **实力等级**：化神期
- **动机**：通过各类骗局收集因果之力，企图突破到大乘期
- **骗局手法**：易容伪装、情感操控、团队作案
- **最终结局**：被林阳揭穿骗局，身败名裂，最终被因果之力反噬而亡

**钱无量**

- **身份**：天机阁副阁主，"仙界高回报理财项目"的幕后黑手
- **年龄**：500岁
- **性格特点**：精于算计、表面和善、内心冷酷
- **与主角的冲突**：嫦小娥被骗事件的幕后推手，利用情报优势设计金融骗局
- **实力等级**：元婴期
- **动机**：积累财富购买"天道认证资质"，企图控制整个修真界的情报网络
- **骗局手法**：金融诈骗、情报操控、虚假宣传
- **最终结局**：被林阳揭露罪行，天机阁被解散，钱无量被囚禁

**天道占据者（终极BOSS）**

- **身份**：占据天道本体的神秘存在
- **真实身份**：上古时期的邪神，被封印后通过特殊手段占据了天道
- **性格特点**：深不可测、布局深远、视众生为棋子
- **与主角的冲突**：整个修真界骗局的终极幕后黑手，林阳的终极敌人
- **实力等级**：超越渡劫期（伪天道级别）
- **动机**：通过散布骗局收集因果之力，彻底掌控天道，成为真正的天道主宰
- **骗局手法**：操控因果、幕后布局、借刀杀人
- **最终结局**：被林阳和天道分身联合击败，永远被封印
