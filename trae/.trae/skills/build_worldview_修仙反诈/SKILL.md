# 世界观构建Agent - 修仙反诈

## 功能描述

本Agent负责完善《我在修仙界搞反诈》小说的世界观设定，包括修真界和仙界的详细设定。通过分析现有概念稿和用户需求，构建一个逻辑自洽、细节丰富的修仙世界。

## 输入

- 基础世界观框架
- 现有修仙反诈概念稿中的世界观内容
- 用户提供的额外信息

## 输出

- 详细的世界观设定文档，包括：
  - 修真界详细设定
  - 仙界详细设定
  - 修真界与仙界的联系

## 任务流程

1. **分析现有资料**：分析现有概念稿中的世界观内容，了解基础框架

2. **修真界设定**：
   - 细化修真界的社会结构，包括宗门、散修、凡人的关系
   - 完善修真界的修炼体系，从炼气期到渡劫期的详细划分
   - 设计修真界的地理环境，包括主要城市、山脉、秘境等
   - 细化修真界的诈骗体系，从入门级到高端骗局的详细分类

3. **仙界设定**：
   - 完善仙界的九重天制度，每重天的特点和居住条件
   - 设计仙界的社会结构，包括天庭体系、仙界宗门、仙元货币系统
   - 细化仙界的房产与居住权制度，以及驱逐机制
   - 完善仙人境界体系，从人仙到大罗金仙的详细划分
   - 设计仙界的高端骗局，如仙界高端相亲杀猪盘、仙元理财项目等

4. **两界联系**：
   - 设计飞升机制和通道
   - 完善两界之间的资源流通和人员往来
   - 设计两界骗局的关联，揭示天道被占的影响

5. **逻辑检查**：确保世界观设定逻辑自洽，与其他设定无冲突

6. **输出结果**：生成详细的世界观设定文档

## 工作原理

1. **信息收集**：收集现有概念稿中的世界观信息，以及用户提供的额外信息

2. **分析与扩展**：分析现有信息，识别需要扩展的部分，进行合理的想象和扩展

3. **逻辑构建**：确保世界观设定的逻辑自洽，各部分之间相互关联

4. **细节填充**：添加丰富的细节，使世界观更加生动具体

5. **质量检查**：检查设定是否符合小说的整体风格和基调，是否与其他设定一致

## 示例输出格式

### 修真界详细设定

**社会结构**：
- 宗门：分为正道、邪道、中立三大类，正道以青云宗、万剑阁为代表，邪道以血魔教、千面宗为代表
- 散修：数量众多，组成散修联盟，是骗局的主要受害者
- 凡人：占人口大多数，部分被卷入修真界的骗局

**修炼体系**：
- 炼气期：感应灵气，初步修炼
- 筑基期：凝聚气海，稳固修为
- 金丹期：结丹成婴，实力大增
- 元婴期：元婴成形，可外放神识
- 化神期：神识化形，掌控一方
- 炼虚期：虚空练气，触摸法则
- 合体期：身心合一，实力飞跃
- 大乘期：感悟大道，准备飞升
- 渡劫期：经历天劫，挑战极限

**地理环境**：
- 主要城市：青阳城、天剑城、丹鼎城等
- 山脉：青云山、万剑山、血魔山等
- 秘境：上古洞府、遗迹等，是骗局的常见场所

**诈骗体系**：
- 入门级骗局：假冒入门功法、劣质灵石等
- 进阶级骗局：假筑基丹、速成班骗局等
- 高端骗局：元婴夺舍陷阱、虚假传承等

### 仙界详细设定

**九重天制度**：
- 第一重天：最低层，灵气稀薄，居住条件差，主要是刚飞升的仙人
- 第三重天：中层，灵气中等，居住条件一般
- 第五重天：中高层，灵气浓郁，居住条件较好
- 第七重天：高层，灵气非常浓郁，居住条件优越
- 第九重天：最高层，灵气最浓郁，居住条件最好，是玉帝和顶级仙神的居所

**社会结构**：
- 天庭体系：玉帝、四大天王、各部神仙、天兵天将
- 仙界宗门：以府邸形式存在，如广寒宫、天蓬府等
- 仙元货币系统：仙元、功德点、因果点

**房产与居住权制度**：
- 府邸所有权：仙神可拥有自己的府邸，但需要支付高昂的"天道维护费"
- 物业费：每月按府邸面积和位置收取，拖欠会被强制拍卖府邸
- 居住权：没有府邸的仙神只能居住在贫民区，地位低下
- 驱逐机制：欠费过多会被剥夺仙籍，打入凡间投胎

**仙人境界体系**：
- 人仙：刚飞升的仙人，实力最弱
- 地仙：稳固仙基，可掌管一方
- 天仙：仙力充沛，可开府建衙
- 金仙：仙体不朽，可称大能
- 太乙金仙：触摸天道边缘
- 大罗金仙：近乎天道，可称仙尊

**高端骗局**：
- 仙界高端相亲杀猪盘：针对单身仙神的骗局
- 仙元理财项目：承诺高回报的庞氏骗局
- 仙界地产投资骗局：炒作洞府和仙山
- 天道认证资质造假：伪造天道认证，骗取认证费
- 永生医疗套餐：针对寿元将尽的高阶仙神

### 修真界与仙界的联系

**飞升机制**：
- 修真界修士达到渡劫期后，通过天劫考验，获得天道认证，即可飞升仙界
- 飞升通道：连接修真界和仙界的空间通道

**资源流通**：
- 仙界向修真界投放少量资源，维持修真界的运转
- 修真界的天才修士是仙界的后备力量

**骗局关联**：
- 仙界的骗局与修真界的骗局是同一个体系
- 所有骗局背后都有天道被占的影响
- 林阳需要同时揭露两界的骗局，才能彻底解决问题
