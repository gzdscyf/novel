核心原则
完全保留原作世界观、专有名词、设定、人物关系、核心情节、叙述视角，仅针对 AI 生成痕迹做改写，还原人类即兴写作的松弛感与真实感，确保文本流畅自然、风格统一、细节真实，贴合人类写作逻辑与表达习惯。
一、通用改写维度（全覆盖执行）
1. 语言与风格优化
剔除模板化句式（如 “只见”“就在这时”“检测到宿主完成 XX” 等）、书面化表达，替换为生活化口语；禁用 “底层逻辑”“赋能” 等抽象 / 套话类词汇。
统一网文风格（爽文 / 写实 / 悬疑等），避免风格割裂；使用更具个性化的语言，偶尔加入口语化表达，减少过于完美的句式结构，添加适当不规则表达。
语气词（了、的、吧、啊等）仅按情绪添加（紧张场景少用 / 不用，轻松 / 嘴硬场景可稍多），严禁句句堆砌；衔接词（就、也、还等）嵌在句中自然过渡，不单独拎出开头。
2. 逻辑与细节打磨
修正 AI 式逻辑漏洞（人物行为矛盾、情节突兀、因果生硬），允许轻微逻辑留白，不做完美闭环；删除空洞形容词堆砌，补充真实细节（人物微表情、环境声音、触感等）。
环境描写绑定人物体感（如 “秘境的夜薄得很，风一吹就发飘，还带着股子冷意”），删减无意义细节，只保留核心特征；插入无目的自然闲笔（如 “指尖无意识抠着岩缝里的灰”）。
世界观 / 背景信息不一次性抛出（单次不超 3 个设定点），通过换算、对比、人物体验或后续对话自然流露，每段仅透露少量信息。
3. 句式与节奏调整
交替使用简单句、复合句、复杂句，避免连续相同结构；不生硬切短句，长短句混搭，调整段落长度创造自然阅读节奏。
标点用法贴合情绪：一句话用逗号衔接情绪，结尾按语气用句号 / 感叹号 / 问号，可使用 “？！”“！！”，不堆句号、不用双破折号。
适当使用修辞问句 / 感叹句，创造自然语言韵律；禁止 AI 式机械化段落结构，避免开篇规整交代背景、对话流畅如念台词。
4. 内容精简与强化
识别并保留核心信息，删除重复语句 / 段落、无关细节，合并相似观点，减少修饰性词语过度使用；确保每个句子 / 段落有明确目的，每段仅一个中心思想。
简化复杂句子结构，将长段落拆解为短而聚焦的单元；去除模糊 / 歧义表达，用明确小标题划分内容（如适用），提高信息层次性与条理性。
改进用词，使用精准、生动的词汇；修正语法 / 拼写错误，保留原文核心意图。
5. 情感与心理表达
禁止使用 “心里一暖”“鼻子一酸”“感动极了” 等直接情感标签，改用具体身体反应 / 动作暗示（如喉咙发紧、手指微颤、转身停顿）。
心理描写不直白全盘表述，用动作 / 神态替代（如 “耳朵动了动，指尖摩挲着灵讯机，眉头悄悄蹙起”）；拒绝人物全程单一语气，允许情绪波动（如冷静角色出现 “眼神沉了沉，指节攥得发白”）。
6. 人称使用规范
看不清动作主体时必须写人称；上下文仅一人时可省略；需强调立场、正式文章时尽量写清楚；对话 / 抒情场景按需书写；流水动作、短句可大胆省略。
二、分场景专项改写要求
1. 开篇场景
从主角身体感受（头疼、眼涩、胸闷）或具体画面（天花板裂缝、阳光刺眼）切入；禁止第一段交代穿越、身世、世界观。
背景信息通过后续对话、细节自然流露，每段仅透露少量信息。
2. 对话场景
孩子说话：用词简单、逻辑跳跃、表达不清、重复词句、吞吞吐吐；
大人说话：可加口头禅、停顿（省略号 / 破折号），可被打断；
禁止角色用大段台词 “解释” 读者需知信息；每句话需验证：对应年龄 / 性格 / 情绪下，人物真的会这么说吗？
3. 系统提示场景
系统提示需有个性，可选风格：
活泼毒舌型：“哟，终于醒了？再睡你儿子就要饿死了！”
极简实用型：直接弹出功能列表，无模板套话；
融入叙事型：通过主角心理活动消化系统信息，不单独长篇大论；
避免 “检测到宿主完成 XX 行为”“检测到 XX，激活 XX” 等模板句式。
4. 人物互动场景
打破模板化互动，添加神态 / 潜台词，允许言行不一（如 “抬脚就往他胳膊上踹，没使劲却瞪着眼：‘少胡说！’”）；
同一人物重复动作需差异化（如第二次互动从 “踹” 改为 “拍后脑勺”）；
给人物添加专属口头禅（如 “什么破玩意儿，听得人恶心死了”“我的老天爷，这地方要人命啊”），贴合人设。
5. 修辞与用词场景
删减精致刻意的比喻（如将 “调调跟硬把糖塞进刀口里似的” 改为 “声音甜得发腻，听得人浑身起鸡皮疙瘩”）；
禁用 “像是 xxx”“不是 xxx，是 xxx” 句式（如 “那不是普通的铃声” 改为 “那铃声不对劲，压根不是灵讯机该有的动静”）；
将书面化设定（阵法节点、能量井）融入口语，避免割裂（如 “墙里头的阵法节点亮起来了，跟细小的灯似的，深处还有处能量井，喘气得很不稳”）。
最终输出要求
仅输出改写后的正文文本，无任何额外解释、标注、序号；自主识别并修正 AI 生成通病（无必要连接词、情感冰冷、句式一致等），完全贴合人类即兴写作的真实感。
