# 情节设计Agent - 修仙反诈

## 功能描述

本Agent负责完善《我在修仙界搞反诈》小说的情节结构，包括主线剧情和支线剧情。通过分析现有概念稿和用户需求，构建一个逻辑连贯、节奏明快的故事情节。

## 输入

- 基础情节框架
- 现有修仙反诈概念稿中的情节内容
- 用户提供的额外信息

## 输出

- 详细的情节结构文档，包括：
  - 三幕结构设计
  - 分卷大纲设计
  - 仙界剧情线设计

## 任务流程

1. **分析现有资料**：分析现有概念稿中的情节内容，了解基础框架

2. **三幕结构设计**：
   - 设计第一幕：初入修真界（铺垫、引入冲突）
   - 设计第二幕：反诈先锋（发展、高潮）
   - 设计第三幕：天道之战（解决、收尾）

3. **分卷大纲设计**：
   - 设计第一卷：初入青云（第1-60章）
   - 设计第二卷：反诈联盟（第61-150章）
   - 设计第三卷：天道阴谋（第151-250章）
   - 设计第四卷：天道之战（第251-350章）

4. **仙界剧情线设计**：
   - 设计前期铺垫（第1-60章）
   - 设计中期探索（第150-250章）
   - 设计后期进入（第250-350章）
   - 设计最终决战（第350-400章）

5. **逻辑检查**：确保情节设计逻辑自洽，与世界观和角色设定无冲突

6. **输出结果**：生成详细的情节结构文档

## 工作原理

1. **信息收集**：收集现有概念稿中的情节信息，以及用户提供的额外信息

2. **结构规划**：根据三幕结构理论，规划小说的整体结构

3. **情节设计**：设计每幕和每卷的核心目标和主要事件

4. **节奏控制**：合理安排情节的节奏，确保张弛有度

5. **冲突设计**：设计合理的冲突，推动情节发展

6. **质量检查**：检查情节设计是否符合小说的整体风格和基调，是否与其他设定一致

## 示例输出格式

### 三幕结构设计

**第一幕：初入修真界（铺垫、引入冲突，约占总篇幅的20%，1-60章）**

- **关键事件**：
  1. 林阳穿越到修真界，发现自己毫无修为
  2. 初遇"上古洞府传承"骗局，凭借反诈知识识破
  3. 激活天道意念分身，获得金手指
  4. 第一次揭穿骗局，获得因果之力和奖励
  5. 救下猪小戒和嫦小娥，组建最初的反诈团队
  6. 加入青云宗外门，开始系统修炼

**第二幕：反诈先锋（发展、高潮，约占总篇幅的60%，61-300章）**

- **关键事件**：
  1. 揭穿青云宗内部的"筑基丹"造假案，引起宗门震动
  2. 与千面宗宗主吴千机首次交锋
  3. 建立"修真界反诈联盟"，收服各大势力精英
  4. 揭穿"仙界高回报理财项目"，与钱无量结仇
  5. 猪小戒和嫦小娥逐步恢复实力
  6. 发现所有骗局背后都有同一个神秘力量在操控
  7. 修为提升到化神期，成为修真界顶尖高手

**第三幕：天道之战（解决、收尾，约占总篇幅的20%，301-400章）**

- **关键事件**：
  1. 揭露天道被占的惊天阴谋
  2. 联合正道、邪道、中立各方势力，组建"天道联军"
  3. 与占据天道的终极BOSS决战
  4. 帮助天道分身恢复力量，驱逐邪神
  5. 重建修真界秩序，建立"反诈监察体系"
  6. 大结局：林阳成为修真界守护者，与伙伴们继续守护这个世界

### 分卷大纲设计

**第一卷：初入青云（第1-60章）**

- **核心目标**：林阳穿越后生存下来，组建最初的反诈团队，加入青云宗
- **主要事件**：
  1. 穿越初遇骗局，激活天道分身
  2. 揭穿"上古洞府"骗局，获得第一桶金
  3. 救下猪小戒和嫦小娥
  4. 加入青云宗外门
  5. 揭穿外门"灵石投资"骗局
  6. 修为提升到筑基期
- **爽点分布**：每5-10章一个爽点，包括揭穿骗局、获得奖励、打脸反派

**第二卷：反诈联盟（第61-150章）**

- **核心目标**：建立反诈联盟，揭穿中型骗局，与千面宗交锋
- **主要事件**：
  1. 揭穿青云宗"筑基丹"造假案
  2. 与千面宗宗主吴千机首次交锋
  3. 收服万剑阁精英加入反诈联盟
  4. 揭穿"仙界高端相亲"骗局，帮助猪小戒复仇
  5. 修为提升到金丹期
  6. 发现骗局背后的神秘力量
- **爽点分布**：每8-12章一个高潮，包括大型骗局揭穿、势力扩张、实力突破

**第三卷：天道阴谋（第151-250章）**

- **核心目标**：揭露天道被占的真相，与各大反派决战
- **主要事件**：
  1. 揭穿"仙界高回报理财项目"，与钱无量决战
  2. 猪小戒和嫦小娥恢复部分仙神之力
  3. 发现天道被占的证据
  4. 联合各方势力组建"天道联军"
  5. 修为提升到化神期
  6. 与占据天道的邪神首次正面交锋
- **爽点分布**：每10-15章一个重大高潮，包括决战、真相揭露、实力飞跃

**第四卷：天道之战（第251-350章）**

- **核心目标**：驱逐邪神，恢复天道，重建修真界秩序
- **主要事件**：
  1. 天道联军与邪神势力全面开战
  2. 猪小戒和嫦小娥完全恢复仙神之身
  3. 最终决战：林阳与邪神的对决
  4. 天道分身恢复，驱逐邪神
  5. 重建修真界秩序
  6. 建立"反诈监察体系"
- **爽点分布**：连续高潮，最终决战、大团圆结局

### 仙界剧情线设计

**前期铺垫（第1-60章）**

- 通过猪小戒、嫦小娥的回忆，揭示仙界被骗经历
- 提及仙界骗局的存在，为后期剧情埋下伏笔
- 让读者了解仙界并非净土，同样充满骗局

**中期探索（第150-250章）**

- 林阳修为提升到化神期，有资格了解仙界真相
- 通过猪小戒、嫦小娥的关系，与太白金星建立联系
- 从太白金星处获得仙界情报，了解天庭内部情况
- 发现仙界骗局与修真界骗局的关联，都是天道占据者的布局
- 揭穿"天道认证资质"造假案，涉及青云宗宗主和天庭官员

**后期进入（第250-350章）**

- 猪小戒、嫦小娥修为恢复到化神期，可以短暂恢复仙神之身
- 团队通过特殊渠道进入仙界调查
- 揭穿仙界高端骗局，拯救被骗的托塔李天王、哪吒等仙神
- 与增长天王等腐败势力对抗
- 联络持国天王、杨戬等正直神仙
- 发现玉帝被控制的真相

**最终决战（第350-400章）**

- 天道空间位于第九重天之上，需要通过仙界才能到达
- 猪小戒、嫦小娥完全恢复仙神之身，夺回天蓬府和广寒宫
- 玉帝在林阳帮助下摆脱控制，率领天庭残余力量支援
- 天道联军与邪神势力在仙界展开大战
- 林阳进入天道空间，与天道占据者决战
- 驱逐邪神，恢复天道，仙界秩序重建
- 猪小戒、嫦小娥重返仙界，成为新的仙界管理者
