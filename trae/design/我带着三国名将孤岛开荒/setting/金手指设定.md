# 金手指设定

## 一、核心金手指：三国英灵召唤系统

### 1.1 系统来源

主角穿越前通关《三国群英传》，穿越时携带游戏的"因果"，与龟岛英灵殿产生共鸣，获得召唤沉睡在英灵殿的三国英灵的能力。

### 1.2 英灵殿的真相

千百年前，玄龟在沧澜界边缘沉睡。不知何故，三国时期的英雄豪杰死后，战魂被玄龟吸入龟壳，沉睡在英灵殿。每个英灵都有自己的雕像，雕像里寄存着他们的战魂。

英灵沉睡期间：

- 记忆保留，但身体定格在"青年期"（他们战死或成名前的状态）
- 不知道自己后来的事迹（比如关羽不知道自己后来水淹七军、败走麦城）
- 有完整的人格和独立意识

### 1.3 系统界面

```
【三国英灵召唤系统】

宿主：林阳
当前灵蕴：150点

【召唤选项】
- 普通召唤（50灵蕴）- 随机召唤
- 高级召唤（100灵蕴）- 出名英灵概率更高
- 定向召唤（150灵蕴）- 指定阵营（魏/蜀/吴/群英）
- 指定召唤（200灵蕴）- 召唤之前收服失败的英灵

【已收服英灵】
- 暂无

【收服失败列表】
- 暂无
```

## 二、召唤机制

### 2.1 召唤类型

| 召唤类型     | 消耗灵蕴 | 说明                                          | 适用场景               |
| ------------ | -------- | --------------------------------------------- | ---------------------- |
| **普通召唤** | 50灵蕴   | 随机召唤一个英灵，可能是普通/出名英灵         | 初期灵蕴少，先试试运气 |
| **高级召唤** | 100灵蕴  | 随机召唤一个英灵，出名英灵概率更高            | 想要出名英灵           |
| **定向召唤** | 150灵蕴  | 指定阵营（魏/蜀/吴/群英），随机召唤该阵营英灵 | 想要特定阵营英灵       |
| **指定召唤** | 200灵蕴  | 召唤之前收服失败的英灵                        | 收服失败后补救         |

### 2.2 召唤概率

| 召唤类型     | 普通英灵概率 | 出名英灵概率             |
| ------------ | ------------ | ------------------------ |
| **普通召唤** | 70%          | 30%                      |
| **高级召唤** | 40%          | 60%                      |
| **定向召唤** | 50%          | 50%                      |
| **指定召唤** | 0%           | 100%（指定哪个就是哪个） |

### 2.3 英灵阵营

| 阵营     | 代表英灵                                                   |
| -------- | ---------------------------------------------------------- |
| **魏**   | 曹操、荀彧、郭嘉、许褚、典韦、张辽、夏侯惇、夏侯渊、司马懿 |
| **蜀**   | 刘备、关羽、张飞、赵云、诸葛亮、黄忠、马超、魏延、姜维     |
| **吴**   | 孙权、周瑜、鲁肃、甘宁、太史慈、陆逊、吕蒙、黄盖、周泰     |
| **群英** | 吕布、貂蝉、董卓、袁绍、华雄、颜良、文丑、张角、华佗       |

### 2.4 召唤流程

1. 主角通过意识连接英灵殿，选择召唤类型
2. 消耗灵蕴，系统随机/指定英灵
3. **主角直接进入英灵幻境**（英灵的出名历史事件场景）
4. 在幻境中参与英灵的历史事件，体验他们的经历
5. 通过帮助英灵完成历史事件，建立情感联系，获得认可
6. 收服成功后，主角和英灵一起返回龟岛

### 2.5 英灵幻境

英灵幻境是英灵的出名历史事件场景，主角进入后：

| 幻境特点 | 说明 |
| -------- | ---- |
| **时间流速** | 幻境中1天=现实1小时 |
| **主角身份** | 主角以"关键人物"身份出现，参与到名将的历史事件中 |
| **装扮融入** | 主角的穿着会自动融入幻境环境，符合历史背景 |
| **收服方式** | 帮助英灵完成历史事件，建立情感联系，获得认可 |
| **幻境时长** | 根据历史事件的复杂度，幻境体验时间从几天到几个月不等 |
| **收服失败** | 主角被踢出幻境，灵蕴不返还，英灵记录到失败列表 |
| **记忆消除** | 收服成功后，英灵会忘记幻境里的事，只记得对主角的忠诚 |

### 2.5.1 主要英灵的幻境事件

| 英灵 | 历史事件 | 幻境场景 | 考验内容 |
|------|----------|----------|----------|
| **赵云** | 长坂坡之战 | 长坂坡战场 | 帮助赵云单骑救主，突破曹军重围 |
| **诸葛亮** | 草船借箭 | 赤壁之战前 | 协助诸葛亮完成草船借箭，展示智谋 |
| **关羽** | 温酒斩华雄 | 汜水关前 | 与关羽并肩作战，见证温酒斩华雄 |
| **张飞** | 当阳桥断后 | 当阳桥 | 协助张飞据水断桥，吓退曹军 |
| **吕布** | 三英战吕布 | 虎牢关 | 与吕布并肩作战，展现无双战力 |
| **周瑜** | 火烧赤壁 | 赤壁战场 | 协助周瑜策划火烧赤壁，击败曹操 |
| **典韦** | 宛城救主 | 宛城 | 帮助典韦保护曹操突围，展现忠勇 |

### 2.6 收服仪式

收服成功后，英灵会以特殊方式出场：

**出场特效**：
- 金色光芒从天而降
- 英灵从光芒中走出
- 光芒散去，英灵单膝跪地

**臣服台词**（示例）：
- 赵云："常山赵子龙，愿为主公赴汤蹈火，万死不辞！"
- 诸葛亮："亮虽不才，愿为主公鞠躬尽瘁，死而后已！"
- 关羽："关某愿追随主公，肝脑涂地，在所不惜！"

**臣服动作**：
- 单膝跪地，抱拳行礼
- 目光坚定，神情庄重
- 全身散发气势（武将有战意，谋士有智慧光芒）

**例子**：
- 召唤赵云 → 进入长坂坡之战幻境 → 主角穿古装融入环境 → 赵云正在寻找阿斗 → 主角帮助赵云突破曹军重围，成功救回阿斗 → 赵云认可 → 收服成功 → 金光降临 → 赵云单膝跪地："常山赵子龙，愿为主公赴汤蹈火，万死不辞！" → 赵云忘记幻境记忆，死心塌地跟随主角
- 召唤诸葛亮 → 进入草船借箭幻境 → 主角穿古装融入环境 → 诸葛亮正在准备草船 → 主角协助诸葛亮借箭，应对曹操的怀疑 → 诸葛亮认可 → 收服成功 → 金光降临 → 诸葛亮单膝跪地："亮虽不才，愿为主公鞠躬尽瘁，死而后已！" → 诸葛亮忘记幻境记忆，死心塌地跟随主角

### 2.6 收服规则

| 情况         | 结果                                                 |
| ------------ | ---------------------------------------------------- |
| **收服成功** | 英灵加入，无灵蕴奖励，主角和英灵一起返回龟岛         |
| **收服失败** | 灵蕴不返还，英灵记录到"收服失败列表"，以后可指定召唤 |

### 2.6 召唤限制

| 限制               | 说明                                     |
| ------------------ | ---------------------------------------- |
| **灵蕴限制**       | 灵蕴获取有限，不能无限召唤               |
| **英灵状态**       | 英灵为青年期，战力有限，需养成           |
| **英灵忠诚**       | 召唤出来的名将一定忠诚，不会背叛         |
| **英灵是独立的人** | 收服后的英灵在龟岛生活，不是主角的"装备" |

## 三、英灵殿认主

### 3.1 英灵殿位置

英灵殿位于**岛屿中心**的禁地深处，被茂密的原始森林和危险生物守护。

### 3.2 认主过程

1. 主角第一次进入英灵殿
2. 英灵殿感应到主角的《三国群英传》因果
3. 英灵殿认主，与主角绑定
4. 之后**只有主角可以进入英灵殿**（巨龟内部）

### 3.3 认主后的好处

- 主角可以通过**意识连接英灵殿**进行召唤，不需要每次都去英灵殿
- 主角可以通过意识连接查看英灵状态、分配任务、进行英灵养成
- 主角可以进入英灵殿迷宫探索，获取灵蕴

## 三、灵蕴获取

### 3.1 获取途径

| 途径             | 灵蕴数量    | 说明                                                           | 前期可行性 |
| ---------------- | ----------- | -------------------------------------------------------------- | ---------- |
| **探索英灵殿**   | 10-50点/次  | 英灵殿内有灵蕴结晶，可搜索获得；越深处迷宫灵蕴越多，但也越危险 | ✅ 高      |
| **击败敌人**     | 5-20点/个   | 击败野兽、海盗、敌人获得；头目级敌人获得更多                   | ✅ 高      |
| **完成系统任务** | 20-100点/个 | 系统发布主线/支线/日常任务，完成奖励                           | ✅ 高      |
| **特殊事件**     | 不定        | 剧情事件、发现宝藏、龟岛移动等奖励                             | ✅ 中      |
| **资源开发**     | 10-30点/次  | 开发龟岛资源（如挖矿、采集灵草）产生灵蕴；与龟岛融合度相关     | ⚠️ 中      |

### 3.2 灵蕴获取详情

#### 3.2.1 探索英灵殿（初期主要来源）

- **迷宫结构**：
  | 层级 | 名称 | 危险程度 | 灵蕴收益 | 特色内容 |
  |------|------|----------|----------|----------|
  | 第一层 | **虎豹骑训练场** | ⭐低 | 10-20点 | 模拟曹军虎豹骑训练场，有普通野兽/机关 |
  | 第二层 | **八阵图迷阵** | ⭐⭐中 | 20-30点 | 诸葛亮八阵图变种，有机关陷阱/奇门遁甲 |
  | 第三层 | **长坂坡险境** | ⭐⭐⭐高 | 30-50点 | 模拟长坂坡战场，有小BOSS（许褚幻影） |
  | 特殊层 | **三国历史回廊** | - | 特殊奖励 | 随机出现，有历史事件/隐藏英灵线索 |

- **探索流程**：
  1. **入口选择**：系统给出3个入口选择（安全/挑战/赌一把）
  2. **迷宫探索**：遇到随机事件（岔路/机关/幻影对话），不同选择不同结果
  3. **BOSS战**：第三层有许褚幻影，用三国知识应对
  4. **结算**：获得灵蕴、线索、碎片，可能解锁新内容

- **随机事件例子**：
  - **选择岔路**：两条路，一条亮着微光（灵蕴），一条有脚步声（敌人）
  - **触发机关**：地面塌陷，前面有宝箱（直接拿有陷阱，先观察安全）
  - **幻影对话**：关羽幻影问"华容道该放曹操吗？"（回答"放"得额外灵蕴，"杀"被攻击）

- **限制**：
  - 每天只能探索1次
  - 单次探索上限：50灵蕴
  - 可随时返回出口

#### 3.2.2 击败敌人

- **获取方式**：击败野兽、海盗、敌人
- **灵蕴数量**：
  - 普通野兽：5-10点
  - 普通海盗：10-15点
  - 海盗头目：20-50点
  - 小BOSS：50-100点
- **限制**：同一敌人区域击败后需要一段时间刷新

#### 3.2.3 完成系统任务

- **任务类型**：
  - 主线任务：20-100点
  - 支线任务：10-50点
  - 日常任务：5-20点
- **例子**：
  - 主线任务：击败海盗头目 → 奖励100点
  - 支线任务：探索龟岛秘境 → 奖励50点
  - 日常任务：训练英灵 → 奖励10点

### 3.3 灵蕴限制

- **单日获取上限**：200灵蕴/天（避免无限刷）
- **单次探索上限**：50灵蕴/次（英灵殿探索）

### 3.4 灵蕴用途

| 用途         | 消耗        | 说明               |
| ------------ | ----------- | ------------------ |
| **普通召唤** | 50点        | 随机召唤一个英灵   |

## 四、英灵分类

英灵分为**武将**和**谋士**两大类，定位和成长体系完全不同：

| 维度         | 武将                         | 谋士                         |
| ------------ | ---------------------------- | ---------------------------- |
| **核心定位** | 前线作战、冲锋陷阵           | 出谋划策、内政管理           |
| **成长体系** | 武道成长                     | 谋略成长                     |
| **技能类型** | 进攻技能、防御技能、闪避技能 | 策略技能、辅助技能、情报技能 |
| **主要作用** | 战场杀敌、训练士兵           | 制定战略、管理内政           |
| **代表英灵** | 赵云、关羽、张飞             | 诸葛亮、荀彧、郭嘉           |

## 八、英灵性格与互动

### 8.1 英灵性格

| 英灵       | 性格特点         | 互动风格                               |
| ---------- | ---------------- | -------------------------------------- |
| **赵云**   | 忠诚、正直、呆板 | 言听计从，偶尔吐槽主角太谨慎           |

| **关羽**   | 傲气、义气、重情 | 需要主角展现诚意，一旦认可便死心塌地   |
| **张飞**   | 暴躁、直爽、莽撞 | 需要驯服，喜欢喝酒，容易冲动           |
| **吕布**   | 野心、傲慢 | 性格傲慢，但一定忠诚，不会叛变         |

### 8.2 互动机制

1. **日常对话**：英灵会与主角聊天，发表看法
2. **决策建议**：英灵会给出建议，但主角做最终决定
3. **分歧处理**：英灵可能与主角产生分歧，需要说服或妥协
4. **忠诚度变化**：根据主角行为，英灵忠诚度会变化

### 7.3 英灵生活

英灵收服后是**独立活生生的人**，在龟岛生活：

| 项目         | 说明                                       |
| ------------ | ------------------------------------------ |
| **住处**     | 英灵殿附近有小院落，英灵各自居住           |
| **日常活动** | 武将练武、谋士读书、偶尔帮主角做事         |
| **邀请机制** | 主角需要英灵帮忙时，去邀请他们（不是召唤） |
| **拒绝可能** | 忠诚度低时，英灵可以拒绝主角的邀请         |
| **离开可能** | 忠诚度过低或特殊事件，英灵可以离开龟岛     |

**例子**：

- 林阳需要赵云帮忙训练士兵 → 去赵云院落邀请 → 赵云同意（忠诚度80）
- 林阳需要关羽帮忙打仗 → 去关羽院落邀请 → 关羽拒绝（忠诚度30，觉得林阳不够诚意）

## 九、系统升级

### 9.1 升级条件

| 等级     | 条件         | 解锁功能                        | 实用性说明                      |
| -------- | ------------ | ------------------------------- | ------------------------------- |
| **Lv.1** | 初始         | 系统激活，解锁基础功能          | 英灵殿探索、灵蕴查看、英灵列表  |
| **Lv.2** | 收服3个英灵  | 解锁"英灵状态查看" + "英灵任务" | 可查看忠诚度/成长值，可分配任务 |
| **Lv.3** | 收服5个英灵  | 解锁"英灵共鸣" + "英灵装备"     | 战斗力提升，可装备武器防具      |
| **Lv.4** | 收服8个英灵  | 解锁"玄龟控制" + "技能融合"    | 控制巨龟移动，技能可融合创新    |
| **Lv.5** | 收服10个英灵 | 解锁"玄龟成长" + "英灵传承"    | 巨龟长大扩张，能力传承给普通人  |
| **Lv.6** | 收服15个英灵 | 解锁"英灵复活" + "玄龟觉醒"    | 英灵可复活，玄龟完全觉醒        |

### 9.2 功能详解

#### 9.2.1 系统基础功能（Lv.1激活）

- **英灵殿探索**：可进入英灵殿迷宫探索，获取灵蕴
- **灵蕴查看**：可查看当前灵蕴数量
- **英灵列表**：可查看已收服的英灵列表
- **英灵召唤**：消耗灵蕴可召唤英灵（普通/高级/定向/指定）

#### 9.2.2 英灵状态查看（Lv.2解锁）

- **功能**：可查看英灵的忠诚度、成长值、技能进度、心情、潜力值
- **实用性**：了解英灵状态，更好管理关系

#### 9.2.3 英灵任务（Lv.2解锁）

- **功能**：可给英灵分配任务
- **任务类型**：
  - 武将：巡逻、战斗、探索、训练
  - 谋士：制定策略、分析情报、资源管理
- **奖励**：英灵成长值+10、忠诚度+2
- **例子**：给赵云分配"探索龟岛北部" → 赵云成长值+10、忠诚度+2、发现新资源点

#### 9.2.4 英灵共鸣（Lv.3解锁）

- **功能**：两个英灵配合战斗，战力提升30%
- **条件**：两个英灵忠诚度都≥70
- **持续时间**：1小时
- **冷却时间**：1天

#### 9.2.5 英灵装备（Lv.3解锁）

- **功能**：可给英灵装备武器/防具
- **装备来源**：探索英灵殿、击败敌人、打造
- **装备效果**：提升英灵战力
- **例子**：给赵云装备"精钢枪"（攻击力+20%）+"皮甲"（防御力+10%）

#### 9.2.6 玄龟控制（Lv.4解锁）

- **功能**：主角获得玄龟控制权，可控制巨龟移动
- **移动速度**：每天移动10-50公里（视主角精神力而定）
- **用途**：控制巨龟移动到其他海域，入侵其他岛屿/大陆
- **消耗**：每次移动消耗10灵蕴
- **例子**：控制巨龟移动到海盗老巢附近，突然袭击

#### 9.2.7 技能融合（Lv.4解锁）

- **功能**：两个英灵的技能融合，产生新技能
- **条件**：两个英灵忠诚度都≥80
- **消耗**：100灵蕴/次
- **例子**：赵云"百鸟朝凤枪"+关羽"青龙偃月刀"→新技能"龙凤双杀"（伤害+50%）

#### 9.2.8 玄龟成长（Lv.5解锁）

- **功能**：消耗灵蕴让玄龟长大，获得更多土地
- **消耗**：1000灵蕴/次
- **效果**：龟岛面积增加10%，可容纳更多人口和建筑
- **限制**：最多成长10次
- **例子**：消耗1000灵蕴 → 龟岛面积从10平方
