# 东汉online 人设卡

## 一、主角人设卡

### 刘辩（汉少帝）

| 项目 | 内容 |
|------|------|
| **姓名** | 刘辩 |
| **身份** | 汉少帝，大汉天子（目前是傀儡） |
| **年龄** | 14岁 |
| **外貌** | 少年天子，面容清秀，眼神中透着与年龄不符的成熟 |
| **性格** | 谨慎、腹黑、善于利用玩家、有现代人的思维 |
| **背景** | 现代人穿越成刘辩，熟知三国历史，知道自己的悲惨结局 |
| **金手指** | 东汉online系统 |
| **目标** | 活下去→夺权→保住皇位→一统天下 |
| **口头禅** | "朕的江山，全是玩家！" |
| **弱点** | 年龄小、无实权、被何进控制 |
| **成长弧线** | 傀儡皇帝→夺权成功→改革朝政→一统天下 |

#### 性格详解

| 性格特点 | 表现 |
|---------|------|
| 谨慎 | 不轻易暴露自己，善于隐藏真实想法 |
| 腹黑 | 表面顺从何进，暗中布局 |
| 善于利用玩家 | 把玩家当工具人，但也会给玩家好处 |
| 有现代人的思维 | 知道科技的重要性，知道历史的走向 |

#### 与玩家关系

| 关系类型 | 说明 |
|---------|------|
| 表面关系 | 玩家以为刘辩是高智能NPC |
| 实际关系 | 刘辩是真人，玩家是帮手 |
| 互动方式 | 通过系统发布任务、奖励玩家 |
| 情感变化 | 从利用到信任，从工具人到战友 |
