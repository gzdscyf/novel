# 东汉online 任务卡系统

## 一、任务系统概述

### 任务来源

| 来源 | 说明 |
|------|------|
| 系统自动发布 | 根据历史事件自动触发 |
| 刘辩手动发布 | 刘辩根据需要发布任务 |
| 玩家触发 | 玩家行为触发隐藏任务 |

### 任务类型

| 类型 | 说明 | 奖励 |
|------|------|------|
| 主线任务 | 推动剧情发展 | 经验值、金币、稀有物品 |
| 支线任务 | 丰富剧情内容 | 经验值、金币、普通物品 |
| 日常任务 | 每日可完成 | 经验值、金币 |
| 隐藏任务 | 特殊触发条件 | 史诗物品、特殊技能 |
| 紧急任务 | 限时完成 | 双倍奖励 |
