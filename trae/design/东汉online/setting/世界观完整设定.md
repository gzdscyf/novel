# 东汉online 世界观完整设定

## 一、穿越设定

### 穿越背景

| 项目 | 内容 |
|------|------|
| **穿越时间** | 东汉中平六年（公元189年），汉灵帝驾崩，刘辩刚登基 |
| **主角身份** | 刘辩（汉少帝），14岁 |
| **历史走向** | 何进被杀→董卓进京→刘辩被废→刘辩被毒死 |
| **穿越原因** | 现代人熬夜追剧，意外穿越 |

### 历史背景

| 项目 | 内容 |
|------|------|
| **朝代** | 东汉末年 |
| **皇帝** | 刘辩（汉少帝） |
| **实权者** | 何进（大将军） |
| **朝政状况** | 十常侍乱政，外戚专权 |
| **天下局势** | 黄巾起义刚平，天下初定，暗流涌动 |

## 二、系统设定

### 系统名称

**东汉online系统**

### 系统功能

| 功能 | 说明 |
|------|------|
| **召唤玩家** | 从现实世界召唤玩家，玩家以为在玩游戏 |
| **发布任务** | 向玩家发布任务，引导玩家帮助刘辩 |
| **资源转化** | 将游戏资源转化为现实资源（金钱、物品等） |
| **玩家管理** | 管理玩家信息、等级、奖励等 |

### 系统界面（刘辩视角）

| 模块 | 内容 |
|------|------|
| **宿主信息** | 刘辩，大汉天子（傀儡），国力极弱，民心低，军力极弱 |
| **玩家列表** | 已召唤玩家信息、等级、状态 |
| **任务管理** | 已发布任务、进行中任务、已完成任务 |
| **资源管理** | 金币、物品、装备等 |
| **系统商城** | 可购买的特殊物品、技能等 |

### 系统限制

| 限制 | 说明 |
|------|------|
| **玩家数量** | 初始10人，随国力提升可增加 |
| **召唤频率** | 每月可召唤一批新玩家 |
| **任务数量** | 同时进行的主线任务不超过3个 |
| **资源转化** | 有每日上限 |

## 三、玩家设定

### 玩家来源

| 项目 | 内容 |
|------|------|
| **来源世界** | 现实世界（21世纪） |
| **筛选条件** | 18-35岁，对历史感兴趣，对游戏感兴趣，无犯罪记录，无精神疾病 |
| **邀请方式** | 手机收到神秘邀请，点击接受即可进入游戏 |

### 玩家附身机制

| 项目 | 内容 |
|------|------|
| **附身方式** | 玩家附身虚构的东汉人，系统创造身份 |
| **记忆融合** | 不融合，玩家无任何记忆，完全靠自己摸索 |
| **身份限制** | 适度限制，玩家可以适当出格，但不能太离谱 |
| **玩家规模** | 中规模（几百人），核心玩家群+路人玩家 |

### 玩家认知

| 项目 | 内容 |
|------|------|
| **玩家视角** | 以为获得了"东汉穿越系统"，在玩超真实的VR游戏 |
| **NPC认知** | 以为刘辩是高智能NPC，不知道是真人 |
| **玩家误会** | 玩家各种骚操作，刘辩各种吐槽 |

### 玩家等级系统

| 等级 | 名称 | 实力描述 | 升级条件 |
|------|------|---------|---------|
| 1级 | 新手玩家 | 普通人，战斗力弱 | 初始等级 |
| 2级 | 初级玩家 | 可单挑2个普通士兵 | 击杀10个敌人 |
| 3级 | 中级玩家 | 可单挑5个普通士兵 | 击杀50个敌人 |
| 4级 | 高级玩家 | 可单挑10个普通士兵 | 击杀200个敌人 |
| 5级 | 精英玩家 | 可单挑20个普通士兵 | 击杀500个敌人 |
| 6级 | 顶级玩家 | 可单挑50个普通士兵 | 击杀1000个敌人 |
| 7级 | 传说玩家 | 可单挑100个普通士兵 | 击杀5000个敌人 |

## 五、现实世界反馈机制

### 能力反馈

| 能力类型 | 是否可反馈 | 反馈方式 | 限制条件 |
|---------|-----------|---------|---------|
| 战斗技能 | ✅ 可反馈 | 玩家在游戏中学的武艺，现实世界也能用 | 需要达到3级以上才能反馈 |
| 体力/耐力 | ✅ 可反馈 | 玩家在游戏中锻炼的体力，现实世界也能提升 | 需要达到2级以上才能反馈 |
| 知识技能 | ✅ 可反馈 | 玩家在游戏中学的知识（锻造、医术等），现实世界也能用 | 无限制 |
| 指挥能力 | ✅ 可反馈 | 玩家在游戏中锻炼的指挥能力，现实世界也能用 | 需要达到5级以上才能反馈 |

### 物品带回

| 物品类型 | 是否可带回 | 带回方式 | 限制条件 |
|---------|-----------|---------|---------|
| 普通物品 | ❌ 不可带回 | - | - |
| 稀有物品 | ✅ 可带回 | 玩家完成稀有任务获得，可带回现实世界 | 每月限带1件 |
| 史诗物品 | ✅ 可带回 | 玩家完成史诗任务获得，可带回现实世界 | 每季度限带1件 |
| 传说物品 | ✅ 可带回 | 玩家完成传说任务获得，可带回现实世界 | 每年限带1件 |

### 金钱兑换

| 项目 | 内容 |
|------|------|
| **兑换比例** | 1游戏金币=10人民币 |
| **兑换限制** | 每月限兑换1000金币（=10000人民币） |
| **兑换方式** | 系统自动转账到玩家银行账户 |

### 系统惩罚机制

| 惩罚类型 | 触发条件 | 惩罚内容 |
|---------|---------|---------|
| 警告 | 玩家在现实世界轻微违规 | 系统警告 |
| 能力削弱 | 玩家在现实世界中度违规 | 现实世界能力削弱50% |
| 能力剥夺 | 玩家在现实世界严重违规 | 现实世界能力剥夺 |
| 账号封禁 | 玩家在现实世界违法 | 账号永久封禁 |

## 六、玩家招募留存机制

### 玩家招募

| 项目 | 内容 |
|------|------|
| **招募方式** | 系统筛选+定向邀请 |
| **筛选条件** | 18-35岁，对历史感兴趣，对游戏感兴趣，无犯罪记录，无精神疾病 |
| **邀请方式** | 手机收到神秘邀请，点击接受即可进入游戏 |
| **招募频率** | 每月招募一批新玩家 |

### 玩家留存

| 项目 | 内容 |
|------|------|
| **奖励机制** | 完成任务获得经验值、金币、物品 |
| **排行榜** | 等级排行榜、财富排行榜、战功排行榜 |
| **社交系统** | 玩家之间可以组队、聊天、交易 |
| **成就系统** | 完成特定成就获得特殊奖励 |
| **现实反馈** | 能力反馈现实、物品带回、金钱兑换 |

### 玩家睡眠问题

| 项目 | 内容 |
|------|------|
| **离线托管** | 玩家离线后，角色自动托管，继续工作 |
| **时间同步** | 游戏时间与现实时间同步，玩家需要睡觉 |
| **疲劳惩罚** | 玩家长时间在线，角色会疲劳，效率下降 |
| **强制下线** | 连续在线超过16小时，强制下线休息8小时 |

## 七、历史惯性机制

### 历史惯性说明

| 项目 | 内容 |
|------|------|
| **机制名称** | 历史惯性 |
| **机制说明** | 改变历史会触发"历史惯性"，引发意外事件 |
| **触发条件** | 改变重大历史事件 |
| **应对方式** | 玩家干预，化解意外 |

### 历史惯性等级

| 等级 | 说明 | 应对方式 |
|------|------|---------|
| 小型惯性 | 改变小事件，引发小规模意外 | 玩家提前干预，防止意外发生 |
| 中型惯性 | 改变中事件，引发中规模意外 | 玩家持续干预，化解危机 |
| 大型惯性 | 改变大事件，引发大规模意外 | 玩家全力干预，扭转局势 |

### 历史惯性示例

| 改变事件 | 触发惯性 | 玩家应对 |
|---------|---------|---------|
| 何进不死 | 十常侍联合其他势力反扑 | 玩家群殴十常侍，镇压反扑 |
| 董卓不进京 | 其他军阀趁机作乱 | 玩家群殴其他军阀 |
| 刘协不被立 | 袁绍等诸侯以"废长立幼"为由讨伐 | 玩家群殴关东联军 |
