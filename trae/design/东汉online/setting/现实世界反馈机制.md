# 东汉online 现实世界反馈机制

## 一、能力反馈机制

### 能力反馈概述

| 项目 | 内容 |
|------|------|
| **机制名称** | 能力反馈 |
| **机制说明** | 玩家在游戏中学的能力，可以反馈到现实世界 |
| **触发条件** | 玩家在游戏中达到一定等级 |
| **反馈方式** | 系统自动反馈，玩家在现实中也能使用该能力 |

### 可反馈能力列表

| 能力类型 | 是否可反馈 | 反馈方式 | 限制条件 |
|---------|-----------|---------|---------|
| 战斗技能 | ✅ 可反馈 | 玩家在游戏中学的武艺，现实世界也能用 | 需要达到3级以上才能反馈 |
| 体力/耐力 | ✅ 可反馈 | 玩家在游戏中锻炼的体力，现实世界也能提升 | 需要达到2级以上才能反馈 |
| 知识技能 | ✅ 可反馈 | 玩家在游戏中学的知识（锻造、医术等），现实世界也能用 | 无限制 |
| 指挥能力 | ✅ 可反馈 | 玩家在游戏中锻炼的指挥能力，现实世界也能用 | 需要达到5级以上才能反馈 |

### 能力反馈等级

| 能力等级 | 游戏内效果 | 现实效果 |
|---------|-----------|---------|
| 1级 | 基础能力 | 无反馈 |
| 2级 | 进阶能力 | 体力/耐力反馈 |
| 3级 | 高级能力 | 战斗技能反馈 |
| 4级 | 精英能力 | 战斗技能强化 |
| 5级 | 顶级能力 | 指挥能力反馈 |
| 6级 | 传说能力 | 全能力反馈 |
| 7级 | 神话能力 | 全能力强化反馈 |

### 能力反馈示例

| 玩家 | 游戏内能力 | 现实反馈 |
|------|-----------|---------|
| 剑指苍穹 | 禁军武艺（5级） | 现实世界也能使用禁军武艺 |
| 铁血丹心 | 军体拳（5级） | 现实世界也能使用军体拳 |
| 医者仁心 | 中医知识（4级） | 现实世界也能使用中医知识 |
| 铁匠老王 | 锻造技术（4级） | 现实世界也能使用锻造技术 |
| 谋定天下 | 谋略知识（5级） | 现实世界也能使用谋略知识 |

## 二、物品带回机制

### 物品带回概述

| 项目 | 内容 |
|------|------|
| **机制名称** | 物品带回 |
| **机制说明** | 玩家在游戏中获得的稀有物品，可以带回现实世界 |
| **触发条件** | 玩家完成稀有任务获得稀有物品 |
| **带回方式** | 系统自动将物品传送到玩家现实中 |

### 物品等级与带回限制

| 物品类型 | 是否可带回 | 带回方式 | 限制条件 |
|---------|-----------|---------|---------|
| 普通物品 | ❌ 不可带回 | - | - |
| 稀有物品 | ✅ 可带回 | 玩家完成稀有任务获得，可带回现实世界 | 每月限带1件 |
| 史诗物品 | ✅ 可带回 | 玩家完成史诗任务获得，可带回现实世界 | 每季度限带1件 |
| 传说物品 | ✅ 可带回 | 玩家完成传说任务获得，可带回现实世界 | 每年限带1件 |

### 物品带回示例

| 物品名称 | 物品等级 | 物品效果 | 带回限制 |
|---------|---------|---------|---------|
| 精良长剑 | 稀有 | 现实世界可用的精良长剑 | 每月限带1件 |
| 史诗方天画戟 | 史诗 | 现实世界可用的史诗武器 | 每季度限带1件 |
| 传说玉佩 | 传说 | 现实世界可用的传说饰品，有特殊效果 | 每年限带1件 |
| 火药配方 | 特殊 | 现实世界可用的火药配方 | 无限制，但需要玩家自己研究 |
| 造纸术配方 | 特殊 | 现实世界可用的造纸术配方 | 无限制，但需要玩家自己研究 |

### 物品带回流程

| 步骤 | 内容 |
|------|------|
| 第一步 | 玩家在游戏中获得稀有物品 |
| 第二步 | 玩家选择"带回现实" |
| 第三步 | 系统确认玩家本月/本季度/本年带回次数 |
| 第四步 | 系统将物品传送到玩家现实中 |
| 第五步 | 玩家在现实中收到物品 |

## 三、金钱兑换机制

### 金钱兑换概述

| 项目 | 内容 |
|------|------|
| **机制名称** | 金钱兑换 |
| **机制说明** | 玩家在游戏中获得的金币，可以兑换现实货币 |
| **触发条件** | 玩家在游戏中获得金币 |
| **兑换方式** | 系统自动转账到玩家银行账户 |

### 兑换规则

| 项目 | 内容 |
|------|------|
| **兑换比例** | 1游戏金币=10人民币 |
| **兑换限制** | 每月限兑换1000金币（=10000人民币） |
| **兑换方式** | 系统自动转账到玩家银行账户 |
| **兑换时间** | 每月1日自动结算上月可兑换金币 |

### 金钱兑换示例

| 玩家 | 游戏内金币 | 可兑换人民币 |
|------|-----------|-------------|
| 商道至尊 | 5000金币 | 50000人民币（需5个月兑换完） |
| 剑指苍穹 | 2000金币 | 20000人民币（需2个月兑换完） |
| 种田小能手 | 3000金币 | 30000人民币（需3个月兑换完） |

## 四、系统惩罚机制

### 惩罚机制概述

| 项目 | 内容 |
|------|------|
| **机制名称** | 系统惩罚 |
| **机制说明** | 玩家在现实世界违规，会受到系统惩罚 |
| **触发条件** | 玩家在现实世界违规 |
| **惩罚方式** | 根据违规程度，给予不同惩罚 |

### 惩罚等级

| 惩罚类型 | 触发条件 | 惩罚内容 |
|---------|---------|---------|
| 警告 | 玩家在现实世界轻微违规 | 系统警告 |
| 能力削弱 | 玩家在现实世界中度违规 | 现实世界能力削弱50% |
| 能力剥夺 | 玩家在现实世界严重违规 | 现实世界能力剥夺 |
| 账号封禁 | 玩家在现实世界违法 | 账号永久封禁 |

### 违规行为示例

| 违规行为 | 违规程度 | 惩罚类型 |
|---------|---------|---------|
| 在现实中炫耀游戏能力 | 轻微 | 警告 |
| 在现实中用游戏能力欺负他人 | 中度 | 能力削弱 |
| 在现实中用游戏能力犯罪 | 严重 | 能力剥夺+账号封禁 |
| 在现实中泄露游戏秘密 | 严重 | 能力剥夺 |

## 六、现实世界限制机制

### 限制机制概述

| 项目 | 内容 |
|------|------|
| **机制名称** | 现实世界限制 |
| **机制说明** | 玩家在现实世界的能力使用有限制 |
| **触发条件** | 玩家在现实世界使用游戏能力 |
| **限制方式** | 能力强度限制、使用频率限制 |

### 能力强度限制

| 能力等级 | 现实强度 | 限制说明 |
|---------|---------|---------|
| 1-2级 | 无反馈 | - |
| 3级 | 50% | 现实世界能力强度为游戏内的50% |
| 4级 | 70% | 现实世界能力强度为游戏内的70% |
| 5级 | 80% | 现实世界能力强度为游戏内的80% |
| 6级 | 90% | 现实世界能力强度为游戏内的90% |
| 7级 | 100% | 现实世界能力强度与游戏内相同 |

### 使用频率限制

| 能力类型 | 使用频率限制 |
|---------|-------------|
| 战斗技能 | 每天最多使用3次 |
| 知识技能 | 无限制 |
| 指挥能力 | 每天最多使用1次 |

### 身份隐藏限制

| 项目 | 内容 |
|------|------|
| **限制说明** | 玩家在现实世界要隐藏游戏身份 |
| **违反后果** | 系统警告，严重者能力剥夺 |
| **例外情况** | 玩家之间可以互相透露身份 |

## 七、现实世界动力系统

### 玩家现实动力

| 动力类型 | 说明 |
|---------|------|
| 能力提升 | 玩家在游戏中变强，现实世界也变强 |
| 财富积累 | 玩家在游戏中赚钱，现实世界也能变现 |
| 技能学习 | 玩家在游戏中学习技能，现实世界也能用 |
| 社交拓展 | 玩家在游戏中交朋友，现实世界也能互动 |

### 玩家游戏动力

| 动力类型 | 说明 |
|---------|------|
| 历史改写 | 玩家参与历史事件，改写历史 |
| 角色成长 | 玩家角色有成长弧线 |
| 社交互动 | 玩家之间有社交关系 |
| 剧情悬念 | 剧情有悬念，玩家想知道后续 |
