# 东汉online 玩家招募留存机制

## 一、玩家招募机制

### 招募流程

| 步骤 | 内容 |
|------|------|
| 第一步 | 系统筛选符合条件的玩家 |
| 第二步 | 向玩家手机发送神秘邀请 |
| 第三步 | 玩家点击接受邀请 |
| 第四步 | 玩家意识进入游戏，附身虚构东汉人 |
| 第五步 | 玩家开始游戏 |

### 筛选条件

| 条件 | 说明 |
|------|------|
| 年龄 | 18-35岁 |
| 兴趣 | 对历史感兴趣，对游戏感兴趣 |
| 品行 | 无犯罪记录，无精神疾病 |
| 健康 | 身体健康，无重大疾病 |
| 时间 | 有足够的空闲时间 |

### 邀请方式

| 项目 | 内容 |
|------|------|
| **邀请形式** | 手机收到神秘APP邀请 |
| **邀请内容** | "你想体验真实的东汉生活吗？点击接受，开启你的穿越之旅！" |
| **接受方式** | 点击"接受"按钮 |
| **拒绝方式** | 点击"拒绝"按钮，邀请失效 |

### 招募频率

| 项目 | 内容 |
|------|------|
| **初始玩家** | 10人 |
| **后续招募** | 每月招募一批新玩家 |
| **招募数量** | 根据国力决定，国力越强，可招募越多 |
| **玩家上限** | 几百人（中规模） |

## 三、玩家睡眠问题解决方案

### 离线托管

| 项目 | 内容 |
|------|------|
| **托管方式** | 玩家离线后，角色自动托管 |
| **托管内容** | 角色继续工作、训练、学习 |
| **托管效率** | 托管效率为在线时的50% |
| **托管限制** | 托管期间不能进行战斗 |

### 时间同步

| 项目 | 内容 |
|------|------|
| **同步方式** | 游戏时间与现实时间同步 |
| **同步比例** | 1:1 |
| **玩家需求** | 玩家需要睡觉，不能24小时在线 |

### 疲劳系统

| 项目 | 内容 |
|------|------|
| **疲劳值** | 玩家有疲劳值，初始为0 |
| **疲劳增加** | 每在线1小时，疲劳值+10 |
| **疲劳效果** | 疲劳值越高，效率越低 |
| **疲劳恢复** | 离线休息，疲劳值恢复 |

#### 疲劳效果表

| 疲劳值 | 效率 |
|--------|------|
| 0-30 | 100% |
| 31-60 | 80% |
| 61-90 | 60% |
| 91-100 | 40% |
| 100+ | 20% |

### 强制下线

| 项目 | 内容 |
|------|------|
| **触发条件** | 连续在线超过16小时 |
| **强制措施** | 强制下线休息8小时 |
| **提示内容** | "您已连续在线16小时，请休息后再来！" |

## 四、玩家现实世界互动

### 现实见面

| 项目 | 内容 |
|------|------|
| **触发条件** | 玩家在现实中相遇 |
| **识别方式** | 系统提示"附近有其他玩家" |
| **互动内容** | 确认彼此身份，组建现实团队 |
| **互动奖励** | 团队buff、特殊技能 |

### 现实组队

| 项目 | 内容 |
|------|------|
| **组队方式** | 玩家在现实中组队 |
| **组队奖励** | 团队buff，游戏内效率提升 |
| **组队限制** | 团队人数不超过10人 |

### 现实冲突

| 项目 | 内容 |
|------|------|
| **冲突类型** | 玩家在现实中发生冲突 |
| **冲突原因** | 利益纠纷、意见不合等 |
| **冲突解决** | 游戏内PK、现实调解 |
| **冲突惩罚** | 违规玩家受到系统惩罚 |

## 五、玩家流失预防

### 流失预警

| 项目 | 内容 |
|------|------|
| **预警条件** | 玩家连续7天未登录 |
| **预警措施** | 发送提醒消息，提供回归奖励 |

### 回归奖励

| 项目 | 内容 |
|------|------|
| **回归条件** | 连续7天未登录后首次登录 |
| **回归奖励** | 500金币、经验值加成卡（3天） |

### 活动激励

| 活动类型 | 活动内容 | 活动奖励 |
|---------|---------|---------|
| 节日活动 | 节日期间特殊活动 | 限定物品、大量金币 |
| 周年活动 | 游戏周年庆典 | 稀有物品、特殊称号 |
| 新版本活动 | 新版本上线活动 | 新内容体验、奖励 |
