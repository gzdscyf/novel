# 东汉online 力量体系与第四天灾机制

## 一、力量体系：低武世界

### 1.1 武将等级

**设定原则**：武将有力气、有武艺，但不是玄幻，不能飞天遁地、不能法术

#### 武将等级划分

| 等级 | 名称 | 实力描述 | 代表人物 |
|------|------|---------|---------|
| 1级 | 普通武将 | 可单挑10个普通士兵 | 普通校尉、都尉 |
| 2级 | 精锐武将 | 可单挑50个普通士兵 | 华雄、颜良、文丑 |
| 3级 | 顶级武将 | 可单挑100个普通士兵 | 张辽、徐晃、张郃 |
| 4级 | 超一流武将 | 可单挑200个普通士兵 | 关羽、张飞、赵云、马超、黄忠 |
| 5级 | 天下第一 | 可单挑300个普通士兵 | 吕布 |

**武将能力**：
- 力量：可以举起几百斤重物
- 速度：可以快速移动，但不是瞬移
- 耐力：可以连续战斗几个时辰
- 武艺：精通各种兵器，有独门绝技
- 骑术：可以骑马冲锋，马术精湛

**武将弱点**：
- 不能飞天遁地
- 不能法术
- 不能刀枪不入
- 会被围攻致死
- 会被陷阱暗算
- 会被火药炸死

### 1.2 士兵等级

| 等级 | 名称 | 实力描述 | 代表 |
|------|------|---------|------|
| 1级 | 普通士兵 | 普通农民征召，战斗力弱 | 郡国兵、县兵 |
| 2级 | 正规士兵 | 经过训练，战斗力一般 | 禁军、边军 |
| 3级 | 精锐士兵 | 久经沙场，战斗力强 | 西凉铁骑、并州狼骑 |
| 4级 | 特种士兵 | 精选精锐，战斗力极强 | 虎豹骑、陷阵营 |

### 1.3 玩家等级

**设定原则**：玩家初始很弱，但可以升级，最终可以群殴武将

#### 玩家等级划分

| 等级 | 名称 | 实力描述 | 升级条件 |
|------|------|---------|---------|
| 1级 | 新手玩家 | 普通人，战斗力弱 | 初始等级 |
| 2级 | 初级玩家 | 可单挑2个普通士兵 | 击杀10个敌人 |
| 3级 | 中级玩家 | 可单挑5个普通士兵 | 击杀50个敌人 |
| 4级 | 高级玩家 | 可单挑10个普通士兵 | 击杀200个敌人 |
| 5级 | 精英玩家 | 可单挑20个普通士兵 | 击杀500个敌人 |
| 6级 | 顶级玩家 | 可单挑50个普通士兵 | 击杀1000个敌人 |
| 7级 | 传说玩家 | 可单挑100个普通士兵 | 击杀5000个敌人 |

**玩家能力**：
- 现代知识：火药、造纸、印刷、军事战术、经济理论
- 团队协作：几百玩家可以群殴武将
- 无限复活：死亡后24小时复活，可以用人海战术
- 装备系统：可以获得装备，提升战斗力
- 技能系统：可以学习技能，提升战斗力

**玩家限制**：
- 初始很弱，需要升级
- 单挑打不过武将
- 需要团队协作
- 死亡会掉装备和经验

## 二、第四天灾机制

### 2.1 玩家召唤机制

#### 召唤条件

| 条件 | 说明 |
|------|------|
| 召唤上限 | 初期100人，随着刘辩势力扩大，上限增加 |
| 召唤方式 | 系统随机抽取符合条件的现代玩家 |
| 召唤时间 | 每天可召唤10人，累计不超过上限 |
| 召唤限制 | 玩家必须同意穿越，不能强制召唤 |

#### 召唤流程

1. 系统筛选符合条件的玩家
2. 向玩家发送"东汉穿越系统"邀请
3. 玩家同意后，进入东汉世界
4. 玩家以为获得了穿越系统，不知道刘辩是真人

#### 玩家筛选标准

| 标准 | 说明 |
|------|------|
| 年龄 | 18-35岁 |
| 语言 | 中文 |
| 兴趣 | 对历史、游戏、穿越感兴趣 |
| 品行 | 无严重犯罪记录 |

### 2.2 任务系统

#### 任务类型

| 类型 | 说明 | 奖励 |
|------|------|------|
| 主线任务 | 推动剧情发展 | 大量经验、装备、金钱 |
| 支线任务 | 辅助主线发展 | 中量经验、装备、金钱 |
| 日常任务 | 每日可重复 | 少量经验、金钱 |
| 活动任务 | 限时任务 | 特殊奖励 |
| 紧急任务 | 突发事件 | 高额奖励 |

#### 任务示例

**主线任务：群殴何进**
- 任务描述：何进想控制刘辩，击败何进
- 任务目标：击杀何进
- 任务奖励：经验+1000，金钱+10000，装备【精良长剑】
- 任务时限：7天

**支线任务：收服吕布**
- 任务描述：吕布勇猛无敌，收服吕布
- 任务目标：让吕布归顺刘辩
- 任务奖励：经验+5000，金钱+50000，装备【史诗方天画戟】
- 任务时限：30天

**日常任务：巡逻洛阳**
- 任务描述：巡逻洛阳城，维护治安
- 任务目标：巡逻2小时
- 任务奖励：经验+100，金钱+1000
- 任务时限：每日

**紧急任务：董卓进京**
- 任务描述：董卓率军进京，拦截董卓
- 任务目标：击败董卓先锋军
- 任务奖励：经验+2000，金钱+20000，装备【稀有铁甲】
- 任务时限：3天

### 2.6 玩家发展路线

#### 战斗型玩家

**发展路线**：
1. 初期：学习基础武艺，参与群殴
2. 中期：学习进阶武艺、骑术、射术，成为精锐战士
3. 后期：学习大师武艺、指挥，成为玩家军团指挥官

**代表玩家**：剑指苍穹（退伍军人，特种兵）

**发展方向**：
- 单挑流：学习各种武艺，单挑敌人
- 指挥流：学习指挥，带领玩家军团
- 骑兵流：学习骑术，成为骑兵
- 弓箭流：学习射术，成为弓箭手

#### 种田型玩家

**发展路线**：
1. 初期：学习基础农业知识，种植粮食
2. 中期：学习进阶农业知识、锻造、烹饪，发展经济
3. 后期：学习火药、造纸、印刷、钢铁，科技大爆发

**代表玩家**：种田小能手（农业专家，博士）

**发展方向**：
- 农业流：种植粮食、养殖牲畜
- 工业流：锻造装备、制作器械
- 科技流：研究火药、造纸、印刷、钢铁
- 后勤流：烹饪、急救、补给

#### 商人型玩家

**发展路线**：
1. 初期：学习基础贸易知识，买卖商品
2. 中期：学习进阶贸易知识、经济理论，发展商业
3. 后期：学习高级经济理论，经济战

**代表玩家**：商道至尊（商人，企业家）

**发展方向**：
- 贸易流：买卖商品，赚取差价
- 经济流：发展经济，搞垮敌人经济
- 情报流：收集情报，分析局势
- 外交流：外交谈判，结盟敌人

#### 谋士型玩家

**发展路线**：
1. 初期：学习基础谋略知识，分析局势
2. 中期：学习进阶谋略知识、历史知识，出谋划策
3. 后期：学习高级谋略知识，制定战略

**代表玩家**：谋定天下（历史爱好者，三国迷）

**发展方向**：
- 战略流：制定战略，规划大局
- 战术流：制定战术，指挥战斗
- 谋略流：出谋划策，阴谋诡计
- 分析流：分析局势，预测走向

#### 咸鱼型玩家

**发展路线**：
1. 初期：摸鱼、看戏、聊天
2. 中期：继续摸鱼、看戏、聊天
3. 后期：还是摸鱼、看戏、聊天

**代表玩家**：摸鱼达人（大学生，游戏宅）

**发展方向**：
- 气氛组：调节气氛，搞笑担当
- 吐槽组：吐槽剧情，吐槽玩家
- 看戏组：看热闹不嫌事大
- 摸鱼组：佛系玩家，随缘发展

## 三、历史惯性机制

### 3.1 天道反噬

**设定原则**：改变历史会引发"天道反噬"，需要玩家群殴化解

#### 反噬类型

| 类型 | 说明 | 化解方式 |
|------|------|---------|
| 小型反噬 | 改变小事件，引发小规模天灾 | 玩家群殴天灾 |
| 中型反噬 | 改变中事件，引发中规模天灾 | 玩家群殴天灾 |
| 大型反噬 | 改变大事件，引发大规模天灾 | 玩家群殴天灾+刘辩祭天 |

#### 反噬示例

**小型反噬：救下何进**
- 历史走向：何进被十常侍所杀
- 改变事件：玩家救下何进
- 反噬：洛阳城内突发小规模地震
- 化解：玩家群殴地震（用现代技术加固建筑）

**中型反噬：阻止董卓进京**
- 历史走向：董卓进京，废刘辩
- 改变事件：玩家阻止董卓进京
- 反噬：洛阳城外突发洪水
- 化解：玩家群殴洪水（用现代技术治水）

**大型反噬：刘辩不死**
- 历史走向：刘辩被毒死
- 改变事件：玩家保护刘辩，刘辩不死
- 反噬：天下大旱，蝗灾四起
- 化解：玩家群殴蝗灾+刘辩祭天（刘辩假装祭天，实际是玩家搞的）

### 3.2 历史修正力

**设定原则**：历史会自动修正，玩家需要持续干预

#### 修正类型

| 类型 | 说明 | 应对方式 |
|------|------|---------|
| 人物修正 | 历史人物会自动走向历史命运 | 玩家持续干预 |
| 事件修正 | 历史事件会自动发生 | 玩家持续干预 |
| 势力修正 | 历史势力会自动形成 | 玩家持续干预 |

#### 修正示例

**人物修正：吕布**
- 历史命运：吕布会背叛丁原、董卓
- 修正表现：吕布会自动产生背叛念头
- 应对方式：玩家持续给吕布好处，让他不想背叛

**事件修正：官渡之战**
- 历史命运：曹操会在官渡击败袁绍
- 修正表现：曹操会自动获得优势
- 应对方式：玩家提前干预，让袁绍获得优势

**势力修正：三国鼎立**
- 历史命运：魏蜀吴三国鼎立
- 修正表现：天下会自动分裂成三股势力
- 应对方式：玩家提前干预，阻止分裂
