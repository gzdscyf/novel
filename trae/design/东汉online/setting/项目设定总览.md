# 东汉online 项目设定总览

## 项目基本信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 东汉online：朕的江山全是玩家 |
| 目标平台 | 番茄小说 |
| 总章节数 | 500章 |
| 预计字数 | 125万字（每章2500字） |
| 创作时间 | 2026-04-08 |

---

## 一、核心设定

### 1. 穿越设定

| 项目 | 内容 |
|------|------|
| 穿越时间 | 东汉中平六年（公元189年），汉灵帝驾崩，刘辩刚登基 |
| 主角身份 | 刘辩（汉少帝），14岁 |
| 历史走向 | 何进被杀→董卓进京→刘辩被废→刘辩被毒死 |

### 2. 玩家附身机制

| 项目 | 内容 |
|------|------|
| 附身方式 | 玩家附身虚构的东汉人，系统创造身份 |
| 记忆融合 | 不融合，玩家无任何记忆，完全靠自己摸索 |
| 身份限制 | 适度限制，玩家可以适当出格，但不能太离谱 |
| 玩家规模 | 中规模（几百人），核心玩家群+路人玩家 |

### 3. 玩家认知

| 项目 | 内容 |
|------|------|
| 玩家视角 | 以为获得了"东汉穿越系统"，在玩超真实的VR游戏 |
| NPC认知 | 以为刘辩是高智能NPC，不知道是真人 |
| 玩家误会 | 玩家各种骚操作，刘辩各种吐槽 |

### 4. 现实世界反馈

| 项目 | 内容 |
|------|------|
| 能力反馈 | 玩家在游戏中学的能力，现实世界也能用（需达到一定等级） |
| 物品带回 | 玩家可以带回稀有/史诗/传说物品（有数量限制） |
| 金钱兑换 | 游戏金钱可按比例兑换现实货币（1金币=10人民币，每月限1000金币） |

---

## 二、力量体系

### 1. 武将等级

| 等级 | 名称 | 实力描述 | 代表人物 |
|------|------|---------|---------|
| 1级 | 普通武将 | 可单挑10个普通士兵 | 普通校尉、都尉 |
| 2级 | 精锐武将 | 可单挑50个普通士兵 | 华雄、颜良、文丑 |
| 3级 | 顶级武将 | 可单挑100个普通士兵 | 张辽、徐晃、张郃 |
| 4级 | 超一流武将 | 可单挑200个普通士兵 | 关羽、张飞、赵云、马超、黄忠 |
| 5级 | 天下第一 | 可单挑300个普通士兵 | 吕布 |

### 2. 玩家等级

| 等级 | 名称 | 实力描述 | 升级条件 |
|------|------|---------|---------|
| 1级 | 新手玩家 | 普通人，战斗力弱 | 初始等级 |
| 2级 | 初级玩家 | 可单挑2个普通士兵 | 击杀10个敌人 |
| 3级 | 中级玩家 | 可单挑5个普通士兵 | 击杀50个敌人 |
| 4级 | 高级玩家 | 可单挑10个普通士兵 | 击杀200个敌人 |
| 5级 | 精英玩家 | 可单挑20个普通士兵 | 击杀500个敌人 |
| 6级 | 顶级玩家 | 可单挑50个普通士兵 | 击杀1000个敌人 |
| 7级 | 传说玩家 | 可单挑100个普通士兵 | 击杀5000个敌人 |

## 三、分卷大纲

| 卷名 | 核心目标 | 章节数 | 主要事件 |
|------|---------|--------|---------|
| 第一卷：洛阳风云 | 活下去，夺权，保住皇位 | 100章 | 穿越觉醒→玩家融入→群殴何进→群殴董卓→收服吕布→改革朝政→应对关东联军 |
| 第二卷：北方争霸 | 击败关东联军，统一北方 | 100章 | 群殴关东联军→收服袁绍→收服曹操→西征马腾→北征公孙瓒→科技大发展 |
| 第三卷：西征南讨 | 西征益州，南征荆州 | 100章 | 西征刘焉→收服刘焉→南征刘表→收服刘表→平定西南→科技再升级 |
| 第四卷：东征江东 | 东征江东，收服孙坚 | 100章 | 东征孙坚→收服孙坚→平定东南→应对外敌→玩家现实冲突 |
| 第五卷：一统天下 | 一统天下，建立新朝 | 100章 | 建立新朝→改革天下→对外扩张→玩家现实揭秘→大结局 |

## 四、核心爽点

| 爽点类型 | 说明 |
|---------|------|
| 玩家融入趣事 | 玩家融入东汉社会，各种趣事、骚操作 |
| 群殴历史人物 | 玩家群殴何进、董卓、袁绍、曹操、孙坚等 |
| 收服历史人物 | 玩家收服吕布、袁绍、曹操、马腾、刘焉、刘表、孙坚等 |
| 科技碾压 | 玩家搞火药、造纸、印刷、钢铁、蒸汽机等 |
| 现实世界反馈 | 玩家发现能力可以反馈现实，震惊兴奋 |
| 玩家现实互动 | 玩家在现实世界互动、组队、冲突、合作 |
| 国力大增 | 主角国力大增，天下震动 |
