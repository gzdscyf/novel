# 世界观设定

## 一、世界背景

### 1.1 世界名称
**沧澜界**——以海洋为主的世界，岛屿众多，大陆稀少，海域面积占世界90%以上。

### 1.2 时代背景
类似明清时期的社会结构，融合玄幻元素：
- 有武道修炼体系
- 有灵力操控体系
- 有海兽、异兽存在
- 科技水平：冷兵器为主，有初级火器（火炮、火枪）

### 1.3 地理格局
- **三大大陆**：东海大陆、西荒大陆、南疆大陆
- **无数岛屿**：从微型岛到大型岛屿不等
- **迷雾海域**：沧澜界边缘，常年迷雾笼罩，常人难以穿越

## 二、龟岛设定（核心亮点）

### 2.1 本质
龟岛是上古神兽**玄龟**的壳，玄龟沉睡亿万年，壳上长出土壤、植被，形成岛屿。

### 2.2 内部结构

| 层级 | 名称 | 内容 | 进入条件 |
|------|------|------|----------|
| 表层 | 龟岛地表 | 汉朝遗民部落、森林、沙滩、山脉 | 所有人可进入 |
| 中层 | 英灵殿 | 龟壳迷宫，寄存三国英灵 | 仅主角可进入（因果绑定） |
| 深层 | 玄龟本体 | 玄龟沉睡之地 | 后期剧情解锁 |

### 2.3 英灵殿位置

英灵殿位于**岛屿中心**，是岛上居民的**禁地**，世代守护，不让任何人进入。

### 2.4 特殊规则

1. **移动岛屿**：龟岛会缓慢移动，位置不固定，外界难以定位
2. **灵蕴资源**：英灵殿内有"灵蕴"，是召唤英灵的消耗品
3. **因果绑定**：主角因穿越时携带《三国群英传》的因果，成为唯一能进入英灵殿的人
4. **英灵沉睡**：千百年前，三国英灵的战魂被玄龟吸入龟壳，沉睡在英灵殿，等待主角召唤
5. **英灵殿认主**：主角第一次进入英灵殿后，英灵殿认主，只有主角可以进入英灵殿（巨龟内部）
6. **意识连接**：主角可以通过意识连接英灵殿进行召唤，不需要每次都去英灵殿

### 2.5 龟岛资源

| 资源 | 用途 | 分布 |
|------|------|------|
| 灵蕴 | 召唤英灵、英灵养成 | 英灵殿内 |
| 玄铁 | 打造武器 | 龟岛山脉 |
| 灵草 | 炼药、治疗 | 龟岛森林 |
| 海盐 | 贸易、食用 | 龟岛海岸 |

## 三、力量体系

### 3.1 武道体系（凡人修炼）

| 等级 | 名称 | 能力描述 |
|------|------|----------|
| 一至九品 | 武者 | 基础武艺，强身健体 |
| 后天 | 后天高手 | 内力初成，可断石裂木 |
| 先天 | 先天高手 | 内力外放，可隔空伤人 |
| 宗师 | 宗师 | 真气护体，百人敌 |
| 大宗师 | 大宗师 | 真气化形，千人敌 |
| 武圣 | 武圣 | 肉身成圣，万人敌 |

### 3.2 灵修体系（操控灵力）

| 等级 | 名称 | 能力描述 |
|------|------|----------|
| 灵徒 | 灵徒 | 感知灵气 |
| 灵师 | 灵师 | 操控灵气，简单法术 |
| 灵将 | 灵将 | 灵气化形，中等法术 |
| 灵帅 | 灵帅 | 灵气成域，高级法术 |
| 灵王 | 灵王 | 灵气化形，可飞行 |
| 灵皇 | 灵皇 | 灵气通天，可呼风唤雨 |

### 3.3 英灵成长体系

英灵是独立活生生的人，通过战斗、训练、资源投入提升个人能力：

**武将成长**：初入武道 → 小有所成 → 登堂入室 → 炉火纯青 → 巅峰造极

**谋士成长**：初窥门径 → 略有小成 → 运筹帷幄 → 决胜千里 → 神机妙算

详见《金手指设定》中的"武将养成"和"谋士养成"。

### 3.4 主角成长体系

主角是普通人，初始无武道/灵修基础，通过**双修武道和灵修**成长。

**修炼方式**：
- 系统辅助修炼（消耗灵蕴加速修炼）
- 英灵指导（赵云教枪法、诸葛亮教谋略等）
- 战斗历练（实战中积累经验）
- 功法修炼（学习功法提升）

**成长规划**：
| 阶段 | 武道境界 | 灵修境界 | 主要成长方式 |
|------|----------|----------|--------------|
| **前期** | 一至九品武者 | 灵徒 | 系统辅助修炼 + 英灵指导 |
| **中期** | 后天高手 | 灵师 | 战斗历练 + 功法修炼 |
| **后期** | 先天高手 | 灵将 | 突破辅助 + 英灵传承 |
| **巅峰** | 宗师/大宗师 | 灵帅/灵王 | 综合成长 |

详见《金手指设定》中的"主角成长体系"。

## 五、核心规则

1. **灵蕴获取**：通过探索英灵殿、击败敌人、完成任务获取
2. **英灵召唤**：消耗灵蕴，进入英灵幻境，体验英灵青年时期的生活
3. **英灵收服**：在幻境中帮助英灵、建立情感联系，获得认可后收服成功
4. **收服仪式**：收服成功后，金光降临，英灵以中二热血的方式向主角臣服
5. **记忆消除**：收服成功后，英灵会忘记幻境里的事，只记得对主角的忠诚
6. **装扮融入**：主角进入幻境后，穿着会自动融入幻境环境
7. **英灵养成**：英灵通过战斗、训练、资源投入提升成长值
8. **主角成长**：主角双修武道和灵修，通过系统辅助修炼、英灵指导、战斗历练成长
9. **玄龟控制**：系统等级Lv.4后，主角可控制巨龟移动，入侵其他地方
10. **玄龟成长**：系统等级Lv.5后，主角可消耗灵蕴让玄龟长大，获得更多土地
11. **玄龟觉醒**：系统等级Lv.6后，玄龟完全觉醒，可主动战斗
12. **穿越真相**：主角穿越与玄龟有关，后期揭开真相

## 六、英灵召唤与收服

### 6.1 召唤流程

1. 主角通过意识连接英灵殿，选择召唤类型
2. 消耗灵蕴，系统随机/指定英灵
3. 主角直接进入英灵幻境（英灵青年时期的生活场景）
4. 在幻境中进入英灵的生活，体验他们的经历
5. 通过帮助英灵、建立情感联系，完成收服
6. 收服成功后，主角和英灵一起返回龟岛

### 6.2 英灵幻境

| 幻境特点 | 说明 |
| -------- | ---- |
| **时间流速** | 幻境中1天=现实1小时 |
| **主角身份** | 主角以"陌生人"身份出现，需要融入英灵生活 |
| **装扮融入** | 主角的穿着会自动融入幻境环境 |
| **收服方式** | 帮助英灵解决困难、建立情感联系、获得认可 |
| **幻境时长** | 根据英灵不同，幻境体验时间从几天到几个月不等 |
| **收服失败** | 主角被踢出幻境，灵蕴不返还，英灵记录到失败列表 |
| **记忆消除** | 收服成功后，英灵会忘记幻境里的事，只记得对主角的忠诚 |

### 6.3 收服仪式

收服成功后，英灵会以特殊方式出场：

**出场特效**：
- 金色光芒从天而降
- 英灵从光芒中走出
- 光芒散去，英灵单膝跪地

**臣服台词**（示例）：
- 赵云："常山赵子龙，愿为主公赴汤蹈火，万死不辞！"
- 诸葛亮："亮虽不才，愿为主公鞠躬尽瘁，死而后已！"
- 关羽："关某愿追随主公，肝脑涂地，在所不惜！"

**臣服动作**：
- 单膝跪地，抱拳行礼
- 目光坚定，神情庄重
- 全身散发气势（武将有战意，谋士有智慧光芒）
