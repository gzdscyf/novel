# 角色设定

## 一、主角

### 基本信息

| 项目 | 内容 |
|------|------|
| **姓名** | 林阳 |
| **年龄** | 25岁（穿越前） |
| **身份** | 穿越前：程序员+三国迷；穿越后：龟岛"天命之人" |
| **外貌** | 普通长相，但眼神沉稳，给人可靠感 |

### 性格特点

| 类型 | 描述 |
|------|------|
| **核心性格** | 苟、谨慎、腹黑、稳扎稳打 |
| **反差性格** | 关键时刻杀伐果断，护短 |
| **缺点** | 过度谨慎有时错失良机，有选择困难症 |
| **记忆点** | 口头禅"稳一手，别浪"；习惯性摸下巴思考 |

### 目标与动机

| 层面 | 内容 |
|------|------|
| **表面诉求** | 安稳种田过日子，不想惹麻烦 |
| **深层诉求** | 不想当棋子，要掌控自己的命运 |
| **终极目标** | 揭开穿越真相，找到回家的路（或决定留下） |

### 成长线

| 阶段 | 状态 | 关键转折 |
|------|------|----------|
| **前期** | 苟在岛上种田发育，召唤第一批英灵 | 被海盗袭击，被迫反击 |
| **中期** | 被迫卷入争斗，开始扩张势力 | 发现外部世界，决定主动出击 |
| **后期** | 争霸天下，揭开龟岛与世界真相 | 面对终极选择 |

### 金手指

**三国英灵召唤系统**：
- 消耗"灵蕴"资源召唤三国英灵
- 英灵为青年期，需养成解锁能力
- 英灵有独立性格，不是工具人

## 二、三国英灵

### 2.1 首批英灵（第一卷）

#### 赵云

| 项目 | 内容 |
|------|------|
| **青年期状态** | 少年枪客，武艺初成但缺乏实战经验，性格青涩 |
| **养成方向** | 枪法大成+忠诚护主 |
| **召唤消耗** | 100灵蕴 |
| **性格** | 忠诚、正直、有点呆板，对主角言听计从 |
| **成长技能** | 龙胆枪法（觉醒）→ 七进七出（凝形）→ 一身是胆（附体）→ 常山赵子龙（化身） |

#### 诸葛亮

| 项目 | 内容 |
|------|------|
| **青年期状态** | 隐居隆中的青年书生，智谋初显但缺乏实战经验 |
| **养成方向** | 阵法+谋略+治国 |
| **召唤消耗** | 150灵蕴 |
| **性格** | 沉稳、智慧、有点腹黑，喜欢考验主角 |
| **成长技能** | 八阵图（觉醒）→ 草船借箭（凝形）→ 空城计（附体）→ 卧龙出山（化身） |

### 2.2 后续英灵（第二卷起）

| 英灵 | 青年期状态 | 养成方向 | 召唤消耗 | 特殊说明 |
|------|------------|----------|----------|----------|
| **关羽** | 逃亡中的青年武者，义气为先 | 武圣之路+统帅能力 | 120灵蕴 | 忠诚度极高 |
| **张飞** | 屠户出身的猛青年，脾气暴躁 | 狂战之力+冲阵先锋 | 100灵蕴 | 需要驯服 |
| **吕布** | 并州少年骑将，野心初露 | 无双战神 | 200灵蕴 | 忠诚度低，需特殊手段收服 |
| **周瑜** | 江东少年才俊，心高气傲 | 水战+火攻+谋略 | 130灵蕴 | 需要折服 |
| **典韦** | 流浪猛士，忠勇无双 | 贴身护卫+死战之力 | 90灵蕴 | 最忠诚的护卫 |

### 2.3 英灵养成机制

1. **成长值**：英灵有成长值，通过战斗、训练、资源投入提升
2. **技能解锁**：成长到一定阶段解锁新能力/技能
3. **性格互动**：英灵有独立性格，会与主角互动，甚至产生分歧
4. **忠诚度**：忠诚度影响英灵的服从程度，忠诚度过低可能叛变

## 五、人设防崩塌规则

1. **主角从不滥杀无辜**，但对付敌人绝不手软
2. **主角的"苟"是谨慎，不是懦弱**，该出手时果断出手
3. **英灵有独立人格**，不能写成工具人
4. **阿蛮不是花瓶**，她有守护族血脉，后期有重要作用
5. **反派有智商**，不能写成无脑送经验的NPC
6. **英灵必须符合三国历史形象**：
   - 关羽：傲气、义气、重情，不写得太窝囊
   - 张飞：暴躁但直爽，不写得太傻
   - 诸葛亮：智慧但沉稳，不写得太妖孽
   - 赵云：忠诚但正直，不写得太工具人
   - 吕布：野心但勇猛，不写得太弱智
