# 《我在修仙界搞反诈》角色设定

## 一、主角设定

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| **姓名** | 林阳 |
| **年龄** | 穿越前28岁，穿越后20岁（身体年龄） |
| **性别** | 男 |
| **身份** | 前反诈警察 → 青云宗外门弟子 → 修真界反诈先锋 |
| **出生地** | 地球华夏国某市 |
| **当前所在地** | 天玄界东域青云州 |

### 1.2 外貌特征

#### 穿越前（地球时期）
- **身高**：180cm
- **体型**：匀称，因警察训练保持较好身材
- **面容**：棱角分明，眼神锐利，常年的警察工作让他自带一股正气
- **发型**：短发，干练
- **着装偏好**：休闲装或警服，喜欢深色系

#### 穿越后（修真界）
- **身高**：180cm（保持不变）
- **体型**：因穿越后身体重塑，更加年轻，但保留了训练痕迹
- **面容**：20岁左右的年轻面容，但眼神中有着与年龄不符的成熟和锐利
- **发型**：黑色长发（修真界习俗），但常扎成马尾，显得干练
- **着装偏好**：
  - 初期：普通修士服饰，灰蓝色调
  - 中期：青云宗外门弟子服饰，青色长袍
  - 后期：独特的"反诈使"服饰，融合现代警服元素（如肩章、腰带）和修真界长袍

#### 标志性特征
- **眼神**：锐利如鹰，仿佛能看穿一切谎言
- **习惯性动作**：思考时会不自觉地用手指敲击桌面（审讯时的习惯）
- **笑容**：平时温和，但面对骗子时会露出让后者毛骨悚然的"职业微笑"

### 1.3 性格特点

#### 核心性格
- **正义感强**：天生的正义感，无法容忍欺骗和不公
- **逻辑缜密**：警察训练养成的逻辑思维，善于分析推理
- **观察力敏锐**：能注意到常人忽略的细节
- **幽默风趣**：用幽默化解紧张气氛，调节团队情绪
- **坚韧不拔**：面对困难从不轻言放弃

#### 优点
1. **专业能力强**：反诈知识丰富，审讯技巧娴熟
2. **心理素质好**：面对危险能保持冷静
3. **善于沟通**：能与各种人打交道，获取信息
4. **学习能力快**：能快速适应新环境，学习新知识
5. **领导力强**：能团结伙伴，带领团队

#### 缺点
1. **过于较真**：有时候对细节过于执着，显得吹毛求疵
2. **对骗子零容忍**：面对骗子容易情绪激动，失去冷静
3. **职业习惯**：偶尔会陷入审讯口吻，让普通人感到压力
4. **责任感过重**：把太多责任扛在自己身上，容易疲惫
5. **现代思维**：有时候用现代思维看修真界问题，会产生冲突

#### 性格成长
- **初期**：刚穿越时有些迷茫，但很快适应，用专业优势立足
- **中期**：建立团队后，学会信任和依靠伙伴，不再单打独斗
- **后期**：成为修真界守护者，性格更加成熟稳重，但初心不改

### 1.4 背景故事

#### 地球时期
林阳出生在一个普通家庭，父亲是中学教师，母亲是护士。从小正义感强，经常帮助被欺负的同学。大学毕业后考入警校，以优异成绩毕业，进入反诈中心工作。

在反诈中心工作五年，破获多起重大电信诈骗案件，被誉为"骗子克星"。最后一次执行任务时，在追捕一个跨国诈骗团伙头目时，被对方开车撞飞，意外穿越。

#### 穿越初期
穿越到天玄界后，发现自己变成了一个20岁的年轻人，身无分文，毫无修为。但很快凭借反诈知识识破了一个"上古洞府"骗局，激活了天道意念分身，获得了在这个世界立足的资本。

#### 成长历程
- **第1-20章**：在修真界生存，揭穿小型骗局，积累第一桶金
- **第21-60章**：加入青云宗外门，救下猪小戒和嫦小娥，组建反诈团队
- **第61-150章**：建立反诈联盟，揭穿中型骗局，修为提升到金丹期
- **第151-300章**：揭露天道被占的真相，与各大反派决战
- **第301-400章**：驱逐邪神，恢复天道，成为修真界守护者

#### 基本信息

| **性别** | 男 |

| **种族** | 仙神（原）→ 猪妖（现） |

#### 外貌特征
- **原形**：一只粉红色的小猪，约30厘米长，圆滚滚的
- **人形**（后期恢复）：魁梧大汉，满脸胡须，但眼神清澈

- **优点**：忠诚可靠、战斗经验丰富、懂得很多天庭秘闻、幽默搞笑

原天庭天蓬元帅，掌管十万天兵，地位显赫。因陷入竞争对手设计的"仙界高端相亲杀猪盘"，被骗光所有仙元资产并欠下巨额债务。无力支付仙界物业费被驱逐，因债务过重投胎为猪形妖兽。

在凡间流浪时被林阳所救，被林阳的正义感和智慧折服，决定跟随林阳。一方面是为了报恩，另一方面也是为了恢复实力，重返仙界复仇。

| 天罡三十六变 | 仙界神通，目前只能使用部分（如变大变小） |
| 九齿钉耙 | 本命法宝，目前被封印中 |
| 天庭秘闻 | 知道很多天庭的秘密和往事 |
| 战斗经验 | 丰富的战斗经验，可以指导林阳 |

- **初期**：弱小的猪妖，只能使用简单的变化术
