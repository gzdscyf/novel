# 《我和仙帝儿子的日常》角色设定

> 创作时间：2026年4月3日

---

## 角色一：主角（父亲）

### 基础信息
| 项目 | 内容 |
|------|------|
| 姓名 | **林默** |
| 年龄 | 32岁 |
| 身份 | 普通凡人、前理工大学副教授（已辞职）、全职奶爸 |
| 外貌 | 戴黑框眼镜，常年穿格子衬衫配牛仔裤，头发微乱，手指有长期做实验留下的细小烫伤疤痕 |

### 性格特点

**表面人设：**
- 温和内敛，不善言辞，有点书呆子气
- 对儿子百依百顺，典型的"猫爸"
- 遇到修真者会下意识往后缩，本能地觉得自己"惹不起"

**隐藏特质：**
- 骨子里有股倔劲——"你们能修仙，我凭什么不能研究修仙？"
- 极度护短，平时怂，儿子受欺负时会爆发惊人勇气
- 理工男的浪漫：用数据、公式、仪器来理解这个"不讲道理"的修真世界

**缺点：**
- 社交能力弱，经常把天聊死
- 遇到无法解释的现象会钻牛角尖
- 对妻子的"普通"身份从未怀疑过（迟钝）

### 核心能力

**灵力探测器（核心发明）：**
- 外形像老式收音机，能检测方圆百米内的灵力波动
- 可以将灵力强度转化为数值显示（1-9999）
- 后期可升级：识别灵力属性、追踪灵力来源、预警危险

**其他发明：**
- "灵力稳定器"——帮助儿子控制暴走的力量
- "隔音结界发生器"——防止儿子修炼时扰民
- 各种"凡人版法器"——用科学原理模拟修真效果

### 成长弧线

**第一阶段（迷茫期）：**
发现儿子异常 → 震惊 → 恐惧 → 接受现实 → 开始研究

**第二阶段（探索期）：**
沉迷发明 → 忽视儿子情感需求 → 父子产生隔阂 → 反思调整

**第三阶段（觉醒期）：**
意识到"凡人也有凡人的价值" → 不再自卑 → 用科学思维走出独特的"凡人修真路"

**第四阶段（成熟期）：**
成为连接修真界与凡人世界的桥梁 → 儿子最信赖的"军师" → 妻子最安心的后盾

### 目标与动机

**表层目标：**
保护儿子平安长大，帮他隐藏身份不被修真界发现

**深层动机：**
证明自己作为凡人的价值——"我没有灵力，但我有脑子。我能用我的方式，站在你们身边。"

### 口头禅
- "这个数据……不太对劲。"
- "让我先做个实验。"
- "儿子，退后，爸爸来。"（虽然没什么战斗力，但气势要有）

### 微反应设计
- 紧张时会推眼镜（推三下代表极度不安）
- 思考时会无意识转笔
- 生气时不会大吼，而是语速变慢、一字一顿

## 角色二：儿子（核心主角）

### 基础信息
| 项目 | 内容 |
|------|------|
| 姓名 | **林小满**（小名：满满） |
| 年龄 | 7岁 |
| 身份 | 小学一年级学生、仙界至尊仙帝"玄霄"转世 |
| 外貌 | 圆脸大眼，头发软趴趴的，笑起来有酒窝，手腕上常年戴着一个褪色的红绳（母亲给的） |

### 性格特点

**表面人设（普通孩童）：**
- 天真烂漫，对世界充满好奇
- 有点小调皮，喜欢恶作剧
- 爱吃甜食，尤其是草莓味的任何东西
- 怕黑、怕打雷、怕一个人睡觉

**隐藏特质（仙帝底蕴）：**
- 直觉准得可怕（其实是潜意识里的仙帝经验）
- 危机时刻会"本能反应"出惊人的战斗意识
- 对"强者"有天生的俯视感（自己不知道原因）
- 偶尔会说出不符合年龄的话，说完自己就忘了

**缺点：**
- 控制不住力量，经常"闯祸"（比如把玩具捏碎、一跳跳上房顶）
- 太好骗，容易被零食收买
- 自尊心强，被说"没用"会偷偷难过很久

### 关于"无记忆但有底蕴"的设定详解

**为什么保留这个设定：**
这是全文最大的反差萌点和剧情发动机。

**具体表现：**
1. **战斗本能：** 遇到危险时身体会"自己动起来"，但小满只会觉得"咦，我怎么躲开了？"
2. **灵力亲和：** 修炼任何功法都一学就会，但他以为是"大家都这样"
3. **识人能力：** 能本能地分辨好人坏人，但说不出为什么
4. **梦境碎片：** 偶尔会梦到一些光怪陆离的场景（前世记忆碎片），醒来后以为是"动画片看多了"
5. **对"仙帝"相关事物的反应：** 听到"玄霄"这个名字会莫名心悸，看到仙帝画像会觉得"有点眼熟"

**不恢复记忆的原因：**
- 转世时自愿封印记忆（伏笔，后期可揭露原因）
- 保持"赤子之心"是修炼某种至高境界的条件
- 作者选择：有记忆的"老怪物装小孩"太常见，无记忆的真小孩更有趣

### 成长弧线

**第一阶段（懵懂期）：**
发现自己"有点不一样" → 困惑 → 爸爸帮我！

**第二阶段（探索期）：**
开始修炼 → 进步神速但自己不知道 → 交到第一个修真界朋友

**第三阶段（困惑期）：**
发现自己和其他小朋友不一样 → "爸爸，我是不是怪物？" → 父亲的回答成为关键转折点

**第四阶段（觉醒期）：**
遭遇重大危机 → 潜意识里的仙帝之力爆发 → 第一次怀疑自己"是不是有什么来历"

**第五阶段（接纳期）：**
知道真相 → 不接受"我是仙帝" → "我只是林小满，是爸爸的儿子" → 用自己的方式定义身份

### 目标与动机

**表层目标：**
想要"控制自己的力量"，不再闯祸让爸爸担心

**深层动机：**
想要变强保护爸爸——"爸爸没有灵力，我要保护他"

**隐藏动机（潜意识）：**
前世未完成的执念（后期揭晓）

### 口头禅
- "爸爸！你看我！"
- "这个好吃，给爸爸留一半~"
- "我才不是小孩子！"（被摸头时）
- "咦？我怎么又……"（力量失控后）

### 微反应设计
- 开心时会原地小跳
- 心虚时会用手指卷头发
- 认真时会抿着嘴、皱着眉
- 委屈时不会哭出声，而是眼眶红红的、吸鼻子

### 与其他角色的互动模式

**与父亲（林默）：**
- 依赖但不盲从，会反驳爸爸"不科学"的理论
- 是爸爸发明的"首席测试员"（经常出bug）
- 父子间的默契：一个眼神就知道对方要干嘛

**与母亲（苏晚晴）：**
- 更粘妈妈，会撒娇
- 对妈妈有本能的敬畏（察觉不到原因的"妈妈很可怕"）
- 妈妈生气时，小满是唯一敢去哄的人

## 角色三：妻子/母亲

### 基础信息
| 项目 | 内容 |
|------|------|
| 姓名 | **苏晚晴** |
| 年龄 | 外表28岁，实际年龄未知（至少300岁以上） |
| 表面身份 | 全职主妇、温柔贤惠的"普通"妻子 |
| 真实身份 | 隐退的修真界顶尖强者，道号"青璃仙子"，曾是大乘期修士，被称为"剑道第一美人" |
| 外貌 | 长发及腰，常穿素色长裙，眉眼温婉，左手无名指上戴着一枚普通的银戒指（实为封印修为的法器） |

### 性格特点

**表面人设（贤妻良母）：**
- 温柔体贴，把家里打理得井井有条
- 说话轻声细语，从不发脾气
- 对丈夫的"奇怪发明"永远表示支持
- 看起来有点"柔弱"，需要保护

**真实性格（青璃仙子）：**
- 骨子里清冷孤傲，曾经目下无尘
- 护短到极致，动她家人者，虽远必诛
- 有轻微的"厌世"情绪，觉得修真界虚伪无趣
- 选择隐退是因为"看透了"

**缺点：**
- 偶尔会因"太强大"而显得"不近人情"（比如不理解凡人的脆弱）
- 对丈夫的迟钝感到无奈又好笑
- 有时会"过度保护"，暗中解决一切威胁

### 隐退原因（核心伏笔）

**表面原因：**
厌倦了修真界的尔虞我诈，想体验凡人的生活

**真实原因（后期揭晓）：**
- 三百年前的一场大战，她亲手斩杀了自己的挚友（对方入魔）
- 发现修真界最大的阴谋，但无力改变，只能选择逃避
- 更重要的伏笔：她与"玄霄仙帝"（儿子的前世）有过一段渊源

### 成长弧线

**第一阶段（旁观期）：**
冷眼旁观凡人的生活，像在看一场戏 → 被林默的"奇怪"吸引

**第二阶段（入局期）：**
嫁给林默 → 生下小满 → 开始真正"投入"这个角色

**第三阶段（挣扎期）：**
过去的敌人找上门 → 想出手但怕暴露 → 看着丈夫用"凡人的方式"解决问题 → 震撼

**第四阶段（觉醒期）：**
不再逃避过去 → 重新拿起剑 → 但这次是为了守护，而不是杀戮

**第五阶段（圆满期）：**
接纳自己的全部身份——既是青璃仙子，也是苏晚晴，是林默的妻子，是小满的母亲

### 目标与动机

**表层目标：**
守护这个家，让儿子平安成长

**深层动机：**
弥补前世的遗憾（与玄霄的渊源）

**隐藏动机：**
在凡人的生活中寻找"道"的真谛——"不修仙，能不能也活得很好？"

### 口头禅
- "老公，吃饭了~"（温柔模式）
- "哦？是吗。"（危险模式，表示有人要倒霉了）
- "小满，妈妈说过多少次了……"（无奈又宠溺）

### 微反应设计
- 真生气时会笑，笑得越温柔越可怕
- 思考时会无意识摩挲银戒指
- 回忆过去时会望向窗外，眼神放空
- 对丈夫撒娇时会微微歪头（她自己没意识到）

### 与丈夫的相遇（背景设定）

**官方版本（林默知道的）：**
图书馆偶遇，她借了一本《育儿百科》，他帮她拿高处的书，然后聊了起来……

**真实版本（苏晚晴知道的）：**
她当时正在追杀一个魔修，魔修逃进图书馆，她追进去，正好撞见林默。
林默以为她在找书，热情地帮她推荐……
她看着这个"毫无灵力却敢跟陌生人搭话"的凡人，莫名觉得有趣。
后来魔修被她在暗处解决，但她决定留下来，"观察"这个奇怪的人。

**更深层的真相（后期揭晓）：**
这不是偶然。林默的体质特殊（虽然无法修炼，但灵魂纯净），是某种上古血脉的后裔。苏晚晴潜意识里被吸引，是因为他的灵魂与"那个人"（玄霄）有相似之处。

## 三人互动关系图

```
        林默（父亲）
           /    \
          /      \
    "被保护者"   "保护者"
        /          \
       /            \
  林小满（儿子） ———— 苏晚晴（母亲）
   "想保护爸爸"      "暗中守护全家"
```

### 核心互动场景示例

**场景一：早餐时间**
- 林默：一边吃饭一边看数据（灵力探测器的读数）
- 苏晚晴：温柔地提醒"吃饭要专心"，同时暗中把儿子不小心外泄的灵力波动压下去
- 林小满：偷偷把不爱吃的青菜夹到爸爸碗里，被妈妈瞪一眼，又夹回去

**场景二：儿子闯祸后**
- 林小满：（眼眶红红）"爸爸，我又把墙弄穿了……"
- 林默：（推眼镜）"没事，正好测试一下我的'自动修复涂料'。"
- 苏晚晴：（表面）"小满，下次要注意哦。"（暗中）手指一弹，把墙彻底修好

**场景三：危机时刻**
- 敌人来袭，林默把儿子护在身后
- 苏晚晴站在最后面，手已经按在银戒指上准备解封
- 林小满突然爆发，把敌人打飞，然后自己懵了："我、我怎么……"
- 林默：（震惊但强装镇定）"儿子，这个……爸爸以后跟你解释。"
- 苏晚晴：（若有所思地看着儿子）"……原来如此。"

## 角色反差感总结

| 角色 | 表面 | 真实 | 反差效果 |
|------|------|------|----------|
| 林默 | 怂、弱、书呆子 | 倔、护短、用科学走出独特道路 | 凡人也有凡人的骄傲 |
| 林小满 | 普通小孩、爱吃糖、怕黑 | 仙帝转世、潜力无限、战斗本能 | 最萌的反差，最强的底蕴 |
| 苏晚晴 | 温柔主妇、需要保护 | 大乘期强者、剑道第一、曾经孤傲 | 最强的盾，最利的剑 |

---

## 创作提示

1. **林默的"凡人视角"是全文基调**：用他的眼睛看修真界，既有笑点也有泪点
2. **小满的"无记忆"是核心设定**：不要让他突然恢复记忆，保持"真小孩"的纯真
3. **苏晚晴的"隐藏"是最大伏笔**：她的过去、她与玄霄的关系，是后期剧情的重要推动力
4. **家庭感是核心**：无论外面多危险，回到家就是一家三口吃饭、斗嘴、互相嫌弃又互相依赖的日常
