# 写作风格规范（严格统一）

## 1. 视角与叙事

- 叙事视角：第三人称
- 叙事节奏：开篇快速切入冲突，每章至少1个小爽点；中期穿插伏笔，节奏张弛有度；后期加速回收伏笔，高潮迭起
- 场景切换：场景转换用短句过渡（如“与此同时，百里之外的黑风寨……”）/ 用空行分隔不同场景 / 避免频繁切换场景（单章不超过3个场景）

## 2. 文字要求

- 语言风格：简洁有力，动作描写突出（爽文）/ 口语化，贴近生活（都市）
- 对话规范：对话简短，符合角色性格（不堆砌长句），主角对话使用主角名字不要用我。
- 描写侧重：侧重动作打斗（玄幻/武侠）/ 侧重环境世界观（科幻/奇幻）/ 侧重社会现实（都市）

## 3. 禁忌与注意

- 禁用词汇：网络烂梗/低俗词汇/不符合时代的现代用语（古言/民国）
- 避免雷区：无脑爽文套路（如随便捡秘籍）/ 圣母心泛滥 / 强行降智反派 / 逻辑漏洞（如科幻中忽略物理规则）
- 格式要求：章节标题简洁（5-8字）/ 对话用“”，心理活动用（）/ 场景描写另起一段 / 打斗场景分点（不用列表，用段落拆分）
